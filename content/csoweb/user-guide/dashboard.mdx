# 대시보드

CSO 정산 포털에 로그인하면 가장 먼저 표시되는 화면입니다. 정산 데이터의 요약 정보를 확인하고, 상세 데이터를 검색하여 엑셀로 다운로드할 수 있습니다.

## 데이터 계층 구조

정산 데이터는 3단계 계층으로 구성됩니다. 이 구조를 이해하면 화면에 표시되는 데이터를 효과적으로 파악할 수 있습니다.

| 계층 | 설명 | 예시 |
|------|------|------|
| **1단계: CSO관리업체** | 정산 엑셀에 기재된 CSO 업체 단위 | ABC영업, XYZ상사 |
| **2단계: 거래처** | 각 CSO관리업체 아래의 거래처(약국, 병원 등) | OO약국, XX병원 |
| **3단계: 정산 행** | 각 거래처별 개별 처방/거래 내역 | 제품명, 수량, 수수료 등 |

화면에서는 이 구조가 그룹핑되어 표시됩니다. CSO관리업체 아래에 거래처가 묶이고, 각 거래처 아래에 개별 정산 행이 나열됩니다. 거래처별로 회색 합계 행이, CSO관리업체별로 파란색 총합계 행이 표시됩니다.

> 일반회원(CSO 업체)은 자사와 매칭된 CSO관리업체의 데이터만 표시됩니다. 다른 업체의 데이터는 볼 수 없습니다.

## 정산월 선택과 데이터 조회

정산 조회 화면 상단의 드롭다운에서 조회할 **정산월**을 선택합니다. 관리자가 업로드한 정산월 목록이 자동으로 표시되며, 정산월을 선택하면 해당 월의 데이터가 자동으로 로드됩니다.

1. 화면 상단의 **정산월** 드롭다운을 클릭합니다.
2. 조회할 정산월(예: 2026-01)을 선택합니다.
3. 선택 즉시 해당 월의 정산 데이터가 테이블에 표시됩니다.

> 정산월 목록에 원하는 월이 없다면 관리자가 아직 해당 월의 데이터를 업로드하지 않은 것입니다. 관리자(영업관리팀)에게 업로드 일정을 문의하세요.

## 요약 통계 카드

정산월을 선택하면 테이블 상단에 3개의 요약 통계 카드가 표시됩니다. 각 카드의 의미는 다음과 같습니다.

| 카드 | 내용 | 활용 |
|------|------|------|
| **총 수량** | 조회 중인 정산 데이터의 전체 거래 수량 합계 | 해당 월에 영업한 총 거래 규모를 파악할 때 |
| **총 금액** | 조회 중인 정산 데이터의 전체 금액 합계 | 해당 월의 총 매출 규모를 확인할 때 |
| **제약수수료 합계** | 세금계산서 발행 금액 (VAT 포함) | 세금계산서 발행 시 청구 금액을 확인할 때 |

> 제약수수료 합계는 세금계산서 발행의 기준 금액입니다. 공지사항(Notice)에 안내되는 세금계산서 발행일을 확인하고, 이 금액으로 세금계산서를 발행하세요.

## 공지사항(Notice)

테이블 상단에 주황색 박스로 공지사항이 표시됩니다. 관리자가 등록한 안내사항으로, 주로 다음 내용이 포함됩니다.

- 세금계산서 발행 기한
- 정산 마감일
- 정산 관련 유의사항

공지사항은 정산 업무에 직접적으로 영향을 주는 중요한 정보이므로 반드시 확인하세요.

## 검색

테이블 상단의 **검색창**에 키워드를 입력하여 데이터를 필터링할 수 있습니다.

- 제품명 또는 거래처명을 입력하고 **Enter** 키를 누르거나 검색 버튼을 클릭합니다.
- 부분 일치 검색이 지원됩니다. 예를 들어 "아스피"를 입력하면 "아스피린"이 포함된 행이 검색됩니다.
- 검색어를 지우면 전체 데이터가 다시 표시됩니다.

## 표시 항목 설정

정산 데이터는 최대 47개 컬럼으로 구성되어 있습니다. 모든 컬럼을 한 번에 표시하면 화면이 복잡해지므로, **체크박스**를 사용하여 필요한 컬럼만 선택적으로 표시할 수 있습니다.

### 사용 방법

1. 테이블 상단의 **표시 항목** 영역에서 체크박스를 확인합니다.
2. 보고 싶은 컬럼의 체크박스를 선택합니다.
3. 보지 않을 컬럼의 체크박스를 해제합니다.
4. 선택 즉시 테이블에 반영됩니다.

### 필수 항목

회색 배지로 표시된 항목은 **필수 항목**으로, 체크박스를 해제할 수 없습니다. 관리자가 설정한 핵심 컬럼이 항상 표시되도록 보장합니다.

> **주의**: 표시 항목 설정은 엑셀 다운로드에도 동일하게 적용됩니다. 체크한 컬럼만 엑셀 파일에 포함되므로, 다운로드 전에 필요한 컬럼이 모두 선택되어 있는지 확인하세요.

## 주요 컬럼 설명

정산 데이터의 47개 컬럼 중 가장 자주 사용하는 주요 컬럼입니다.

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| 처방월 | 처방이 발생한 연월 | 2026-01 |
| 정산월 | 정산 기준 연월 | 2026-02 |
| CSO관리업체 | 정산 엑셀에 기재된 CSO 업체명 | ABC영업 |
| 거래처명 | 의약품 납품 대상 (병원, 약국 등) | OO약국 |
| 제품명 | 거래 대상 의약품 이름 | OO정 10mg |
| 수량 | 해당 건의 거래 수량 | 500 |
| 금액 | 해당 건의 거래 금액 | 2,500,000 |
| 단가 | 의약품 1단위 가격 | 5,000 |
| 제약_수수료 | CSO 업체에 지급되는 제약수수료 | 150,000 |
| 담당_수수료 | 담당자에게 지급되는 수수료 | 50,000 |

> 47개 컬럼 전체 목록은 엑셀 다운로드 시 모든 항목을 체크한 후 확인하거나, 관리자에게 문의하세요.

## 페이지네이션

조회 결과가 50건을 초과하면 테이블 하단에 페이지네이션이 표시됩니다.

- **이전** 버튼: 이전 페이지로 이동합니다.
- **다음** 버튼: 다음 페이지로 이동합니다.
- 현재 페이지 번호와 전체 페이지 수가 함께 표시됩니다.

50건 이하인 경우에는 페이지네이션이 표시되지 않고 전체 데이터가 한 화면에 나열됩니다.

## 월별 합계

상단 메뉴의 **"월별 합계"**를 클릭하면 정산월별 수수료 합계를 한눈에 확인할 수 있습니다.

### 화면 구성

| 영역 | 내용 |
|------|------|
| 요약 카드 | 총 정산월 수, 총 데이터 건수, 수수료 합계 |
| 합계 테이블 | 정산월별 건수와 합계 컬럼 |
| 최하단 행 | 전체 기간 총합계 |

### 표시 스타일

- 가장 최근 정산월은 **파란색 배경 + "최신" 배지**로 강조됩니다.
- 수수료/합계 관련 숫자는 파란색으로 표시되어 금액을 쉽게 식별할 수 있습니다.

### 정산 조회와의 차이

| 구분 | 정산 조회 | 월별 합계 |
|------|----------|----------|
| 경로 | /dashboard | /monthly-summary |
| 데이터 단위 | 건별 (1행 = 1건의 처방/거래) | 월별 (1행 = 1개월 합산) |
| 용도 | 개별 거래 내역 확인 | 월별 추이 파악 |
| 컬럼 수 | 최대 47개 | 정산월, 건수, 합계 |
