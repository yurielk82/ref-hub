# 시작하기

CSO Web 사용을 위한 초기 설정 가이드입니다.

## 로그인

1. 브라우저에서 CSO Web 주소에 접속합니다
2. 이메일과 비밀번호를 입력하여 로그인합니다
3. 최초 로그인 시 비밀번호 변경을 권장합니다

## 화면 구성

로그인 후 대시보드가 표시됩니다. 좌측 사이드바에서 각 메뉴에 접근할 수 있습니다.
