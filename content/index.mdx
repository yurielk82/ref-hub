# Codegear Dev Portal

프로젝트별 사용자/운영자 매뉴얼을 통합 관리하는 레퍼런스 사이트입니다.

## 프로젝트 목록

| 프로젝트 | 설명 | 스택 |
|---------|------|------|
| [CSO Web](/csoweb) | B2B 제약 정산 수수료 조회 시스템 | Next.js 14 + Supabase |
| [KPIS DSR API](/kpis-dsr-api) | 의약품 공급내역 가공 시스템 | Express 5 + React 19 |
| [StudioGo](/studiogo) | 라이브커머스 스튜디오 예약 플랫폼 | Expo + Hono |
| [EV Motor Reliability](/ev-motor-reliability) | 전기차 모터 신뢰성 시험 자동화 | Python + C (DSP) |

## 매뉴얼 구조

각 프로젝트는 두 가지 매뉴얼을 제공합니다:

- **사용자 매뉴얼** — 실제 서비스 사용자를 위한 가이드
- **운영자 매뉴얼** — 배포, 데이터베이스, 모니터링, 문제 해결 등 시스템 관리 가이드
