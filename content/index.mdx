# Ref Hub

프로젝트별 사용자/운영자 매뉴얼을 통합 관리하는 레퍼런스 사이트입니다.

## 프로젝트 목록

| 프로젝트 | 설명 | 스택 | 버전 |
|---------|------|------|------|
| [CSO Web](/csoweb) | B2B 제약 CSO 수수료 정산 조회 시스템 | Next.js 14 + Supabase + Prisma | v0.65.0 |
| [KPIS DSR API](/kpis-dsr-api) | 의약품 공급내역(갑지) 가공·조회 시스템 | Express 5 + React 19 + PostgreSQL | v0.76.4 |
| [StudioGo](/studiogo) | 라이브커머스 스튜디오 예약 플랫폼 | Expo 52 + Hono + Supabase | v1.5.0 |
| [EV Motor Reliability](/ev-motor-reliability) | 전기차 모터 신뢰성 시험 자동화 도구 | Python 3.11 + C (DSP) + SQLite | v0.40.0 |
| [CoreRx](/corerx) | 제약사 매출 KPI 대시보드 (ETL + BI 분석) | Python 3.11 + PostgreSQL + Superset | v0.1.0 |

## 매뉴얼 구조

각 프로젝트는 두 가지 매뉴얼을 제공합니다:

- **사용자 매뉴얼** — 실제 서비스 사용자를 위한 가이드
- **운영자 매뉴얼** — 배포, 데이터베이스, 모니터링, 문제 해결 등 시스템 관리 가이드
