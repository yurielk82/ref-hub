# AI 분석

AI 기반 데이터 분석 기능의 활용법을 안내합니다.

## AI 채팅

대시보드에서 AI에게 질문하여 데이터에 대한 인사이트를 얻을 수 있습니다.

### 사용 방법

1. 대시보드 화면에서 AI 채팅 패널을 엽니다
2. 한국어로 질문을 입력합니다
3. AI가 현재 데이터를 기반으로 분석 결과를 제공합니다

### 질문 예시

- "이번 달 도매 매출 상위 5개 제품은?"
- "전월 대비 CSO 매출이 감소한 업체는?"
- "6개월 흡수율 추이를 분석해줘"

## AI 설정

사이드바에서 **설정 > AI**를 선택합니다.

| 설정 | 설명 |
|------|------|
| **API 키** | Claude 또는 Gemini API 키 입력 |
| **모델 선택** | 사용할 AI 모델 선택 |
| **활성화** | AI 분석 기능 켜기/끄기 |

> **주의**: API 키는 사용자 본인의 것을 사용합니다. 키가 없으면 AI 기능이 비활성화됩니다.
