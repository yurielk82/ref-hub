# 대시보드 개요

CoreRx 대시보드의 전체 레이아웃, 탭 구조, 공통 필터, 크로스필터 동작을 설명합니다.

## 전체 레이아웃

CoreRx는 단일 대시보드 내에 7개 탭으로 구성됩니다. 모든 탭은 상단 탭 바에서 전환할 수 있으며, 각 탭은 독립적인 차트와 테이블로 이루어져 있습니다.

| 영역 | 위치 | 설명 |
|------|------|------|
| 탭 바 | 화면 상단 | 7개 탭 전환 버튼 |
| 필터 바 | 탭 바 아래 | 기간, 제품, 지역, 거래처 공통 필터 |
| 차트 영역 | 화면 중앙 | 탭별 KPI 카드, 차트, 테이블 |

## 7개 탭 구성

| 탭 | 핵심 질문 | 주요 위젯 |
|----|-----------|-----------|
| 경영 요약 | 경영 핵심 숫자 한눈에 | KPI 카드, ERP vs UBIST 추이, 약효군 파이, TOP 거래처 |
| 제품 성과 | 어떤 제품이 성장하나 | 제품 TOP 20, 제품별 월 매출, 약효군별 비중 |
| 거래처/지역 | 매출 집중도 | 영업소별 매출, TOP 거래처, 거래처 상세 테이블 |
| 시장 분석 | 자사 시장 포지션 | MS% 추이, 경쟁사 비교, 시장 성장률 |
| CSO 성과 | CSO별 수익성 | 흡수율, CSO별 처방금액, 정산 현황 |
| 데이터 관리 | 데이터 최신 여부 | 캐시 상태, 회사명 매핑, 업로드 이력 |
| 시스템 관리 | 사용자/목표 관리 | 사용자 목록, 매출 목표, 달성률 |

## 크로스필터 동작

크로스필터(Cross-filter)는 하나의 차트를 클릭하면 같은 탭의 다른 차트가 자동으로 필터링되는 기능입니다.

### 동작 방식

1. 차트에서 특정 항목(막대, 파이 조각 등)을 **클릭**합니다.
2. 같은 탭의 다른 차트가 해당 항목 기준으로 자동 필터링됩니다.
3. 필터를 해제하려면 같은 항목을 다시 클릭하거나, 다른 빈 영역을 클릭합니다.

### 사용 예시

- **경영 요약** 탭에서 약효군 파이 차트의 특정 약효군을 클릭하면, TOP 거래처 차트가 해당 약효군 거래처만 표시합니다.
- **거래처/지역** 탭에서 영업소별 파이 차트의 영업소를 클릭하면, 거래처 테이블이 해당 영업소 거래처로 필터링됩니다.

> **주의**: 크로스필터는 같은 탭 내에서만 동작합니다. 다른 탭으로 전환하면 필터가 초기화됩니다.

## 공통 필터

필터 바에서 사용할 수 있는 공통 필터입니다. 필터를 변경하면 현재 탭의 모든 차트에 적용됩니다.

| 필터 | 설명 | 기본값 |
|------|------|--------|
| 기간(연월) | 분석 대상 연월 선택 | 최근 12개월 |
| 제품 | 특정 제품 또는 약효군 선택 | 전체 |
| 지역(영업소) | 특정 영업소/지역 선택 | 전체 |
| 거래처 | 특정 거래처 선택 | 전체 |

### 필터 사용법

1. 필터 바에서 원하는 필터 항목의 **드롭다운**을 클릭합니다.
2. 목록에서 값을 선택하거나, 검색창에 키워드를 입력하여 검색합니다.
3. 여러 값을 동시에 선택할 수 있습니다 (다중 선택).
4. **적용** 버튼을 클릭하면 차트가 갱신됩니다.
5. 필터를 초기화하려면 해당 필터의 **X** 버튼을 클릭합니다.

## 탭 전환

- 상단 탭 바에서 원하는 탭 이름을 클릭하면 즉시 전환됩니다.
- 탭 전환 시 공통 필터는 유지되지만, 크로스필터는 초기화됩니다.
- 데이터 로딩 중에는 차트 영역에 로딩 스피너가 표시됩니다.

## 금액 단위

대시보드 전체에서 사용하는 금액과 비율의 단위는 다음과 같습니다:

| 유형 | 단위 | 예시 |
|------|------|------|
| 금액 | 백만원 (₩M) | 1,500 = 15억 원 |
| 비율 | 소수점 1자리 (%) | 23.5% |
| 성장률 | 소수점 1자리 (%p) | +2.3%p |
