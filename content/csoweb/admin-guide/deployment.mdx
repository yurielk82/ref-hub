# 배포

CSO Web 배포 및 환경 설정 가이드입니다.

## 환경

| 환경 | URL | 브랜치 |
|------|-----|--------|
| Production | Vercel 프로덕션 도메인 | `main` |
| Preview | Vercel 프리뷰 도메인 | PR 브랜치 |
| Local | `http://localhost:3000` | 개발 브랜치 |

## 환경 변수

Vercel 대시보드 또는 `.env.local`에서 설정합니다.

| 변수 | 설명 |
|------|------|
| `NEXT_PUBLIC_SUPABASE_URL` | Supabase 프로젝트 URL |
| `NEXT_PUBLIC_SUPABASE_ANON_KEY` | Supabase 공개 키 |
| `SUPABASE_SERVICE_ROLE_KEY` | Supabase 서비스 역할 키 (서버 전용) |

## 배포 절차

1. `main` 브랜치에 PR 머지
2. Vercel이 자동 빌드 및 배포
3. 배포 완료 후 프로덕션 URL 확인
