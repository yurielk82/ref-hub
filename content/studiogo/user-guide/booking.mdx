# 예약하기

StudioGo에서 라이브 커머스 스튜디오를 예약하는 전체 과정을 안내합니다.

## 예약 흐름 요약

| 단계 | 설명 | 소요 시간 |
|------|------|----------|
| 1. 스튜디오 검색 | 지역, 날짜, 설비 조건으로 탐색 | 자유 |
| 2. 슬롯 선택 | 원하는 날짜/시간대 선택 | 자유 |
| 3. Hold (임시점유) | 선택한 슬롯을 2분간 잠금 | **2분 제한** |
| 4. 예약 확정 | 결제 또는 확인 버튼으로 확정 | Hold 시간 내 |
| 5. 승인 대기 | 운영자가 예약을 검토/승인 | 운영자 처리 |

## 스튜디오 검색 및 선택

### 검색 화면 진입

홈 화면 하단 탭에서 **검색** 아이콘을 탭하면 스튜디오 목록 화면으로 이동합니다.

### 필터 조건

검색 화면 상단의 필터를 활용하여 원하는 스튜디오를 찾을 수 있습니다.

| 필터 | 설명 | 예시 |
|------|------|------|
| **지역** | 스튜디오 소재지 | 동대문, 남대문, 성수 |
| **날짜** | 예약 희망일 | 2026-03-15 |
| **설비** | 스튜디오에 구비된 장비 | 조명, 행거, 카메라 |
| **가격대** | 시간당 이용 요금 범위 | 5만~10만원 |

### 스튜디오 상세 정보

목록에서 스튜디오를 탭하면 상세 화면이 표시됩니다. 확인할 수 있는 정보는 다음과 같습니다.

- **스튜디오 사진**: 내부 전경, 설비 사진
- **기본 정보**: 주소, 운영 시간, 수용 인원
- **설비 목록**: 조명, 행거, 풀HD 카메라, Wi-Fi 등
- **이용 요금**: 시간당/일일 요금
- **리뷰**: 다른 BJ의 이용 후기
- **가용 슬롯**: 캘린더 형태로 예약 가능 시간대 표시

## 날짜/시간 슬롯 선택

### 슬롯 개념

스튜디오의 예약 가능 시간은 **슬롯(Slot)** 단위로 나뉩니다. 슬롯 간격은 스튜디오 운영자가 설정하며, 일반적으로 1시간 또는 2시간 단위입니다.

### 슬롯 상태

| 상태 | 표시 | 설명 |
|------|------|------|
| **가용** | 파란색 | 예약 가능한 빈 슬롯 |
| **Hold** | 주황색 | 다른 사용자가 임시점유 중 (2분 내 만료 가능) |
| **예약됨** | 회색 | 이미 예약이 확정된 슬롯 |
| **운영 외** | 빗금 | 운영 시간 밖이거나 차단된 슬롯 |

### 선택 방법

1. 상세 화면에서 **예약하기** 버튼을 탭합니다
2. 캘린더에서 원하는 **날짜**를 선택합니다
3. 해당 날짜의 슬롯 목록에서 **가용(파란색)** 슬롯을 탭합니다
4. 연속된 여러 슬롯을 선택하려면 시작 슬롯과 끝 슬롯을 차례로 탭합니다

## Hold (임시점유)

### Hold란?

슬롯을 선택하면 해당 슬롯에 **Hold**가 걸립니다. Hold는 다른 사용자가 같은 슬롯을 동시에 예약하는 것을 방지하기 위한 **2분간의 임시점유** 장치입니다.

### Hold 규칙

- Hold 유지 시간: **2분 (120초)**
- 화면 상단에 남은 시간이 카운트다운으로 표시됩니다
- 2분 내에 예약을 확정하지 않으면 Hold가 자동 해제되며, 슬롯이 다시 가용 상태로 돌아갑니다
- 한 사용자가 동시에 Hold할 수 있는 슬롯 수는 **1건**입니다
- Hold 중 앱을 종료하거나 뒤로 가기를 누르면 Hold가 즉시 해제됩니다

### Hold 만료 시

Hold가 만료되면 다음과 같은 알림이 표시됩니다.

> "임시점유 시간이 만료되었습니다. 다시 슬롯을 선택해주세요."

만료 후 해당 슬롯이 아직 가용 상태라면 다시 선택할 수 있습니다.

## 예약 확정

### 확정 절차

Hold가 유지되는 2분 이내에 다음 절차를 완료해야 합니다.

1. **예약 정보 확인**: 스튜디오명, 날짜, 시간, 요금을 확인합니다
2. **이용 목적 입력** (선택): 방송 내용, 의류 카테고리 등을 기입합니다
3. **예약 확정** 버튼을 탭합니다
4. 확정 완료 화면이 표시되면 예약이 접수된 것입니다

### 확정 후 알림

예약이 접수되면 다음 알림이 발송됩니다.

| 알림 채널 | 내용 | 발송 시점 |
|----------|------|----------|
| **앱 푸시** | "예약이 접수되었습니다" | 즉시 |
| **카카오 알림톡** | 예약 상세 정보 포함 | 즉시 |
| **인앱 알림** | 알림 탭에 기록 | 즉시 |

## 예약 상태

예약은 다음 상태를 순서대로 거칩니다.

| 상태 | 설명 | 사용자 행동 |
|------|------|-----------|
| **대기** | 운영자의 승인을 기다리는 상태 | 취소 가능 |
| **승인** | 운영자가 예약을 승인한 상태 | 취소 가능 (취소 정책 적용) |
| **체크인** | 예약 당일, 스튜디오에 도착하여 체크인한 상태 | 이용 중 |
| **완료** | 이용이 종료된 상태 | 리뷰 작성 가능 |
| **취소** | 사용자 또는 운영자에 의해 취소된 상태 | - |
| **거절** | 운영자가 예약을 거절한 상태 | 사유 확인 가능 |

### 상태별 알림

각 상태 변경 시 앱 푸시와 카카오 알림톡이 자동 발송됩니다.

- **승인** 시: "예약이 승인되었습니다. 당일 체크인을 잊지 마세요!"
- **거절** 시: "예약이 거절되었습니다. 사유: (운영자 메시지)"
- **예약 1시간 전**: "1시간 후 예약이 있습니다. (스튜디오명) (시간)"

## 예약 확인

### 내 예약 목록

하단 탭의 **내 예약** 아이콘을 탭하면 전체 예약 목록을 확인할 수 있습니다.

- **예정**: 승인된 예약 중 아직 이용 전인 건
- **대기중**: 운영자 승인을 기다리는 건
- **지난 예약**: 완료/취소/거절된 건

### 예약 상세

목록에서 항목을 탭하면 상세 정보를 확인할 수 있습니다.

- 스튜디오 정보 및 위치 (지도 연동)
- 예약 일시 및 슬롯
- 현재 상태
- 운영자 연락처
- 취소/변경 버튼 (상태에 따라 활성화)

## 예약 취소

### 취소 가능 조건

| 예약 상태 | 취소 가능 여부 | 비고 |
|----------|-------------|------|
| 대기 | 가능 | 무조건 취소 가능 |
| 승인 | 가능 | 취소 정책에 따라 수수료 발생 가능 |
| 체크인 | 불가 | 이미 이용 시작 |
| 완료 | 불가 | 이미 이용 종료 |

### 취소 절차

1. **내 예약** 화면에서 취소할 예약을 탭합니다
2. 하단의 **예약 취소** 버튼을 탭합니다
3. 취소 사유를 선택합니다 (일정 변경, 스튜디오 변경, 기타)
4. **취소 확인** 버튼을 탭합니다
5. 취소 완료 알림이 표시됩니다

### 취소 정책

취소 정책은 스튜디오별로 다를 수 있으며, 일반적인 기준은 다음과 같습니다.

| 취소 시점 | 수수료 |
|----------|--------|
| 예약일 3일 전까지 | 없음 |
| 예약일 1일 전까지 | 이용 요금의 30% |
| 예약일 당일 | 이용 요금의 50% |
| 노쇼 (미체크인) | 이용 요금의 100% |
