# KPIS DSR API

의약품 공급내역 가공 시스템 (Korean Pharmaceutical Information Service - Drug Supply Report API)

## 개요

| 항목 | 내용 |
|------|------|
| **스택** | Express 5 + React 19 (Vite) + Radix UI |
| **DB** | SQLite |
| **호스팅** | Vercel (프론트) + NHN Cloud PM2 (API) |
| **인증** | Google OAuth |

## 도메인

- **KPIS**: 의약품관리종합정보센터 — 의약품 유통 보고 의무 관리 기관
- **DSR**: Drug Supply Report — 의약품 공급내역 보고서
- **갑지**: 공급자 보고 문서 (21열)
- **을지**: 일련번호 문서 (3열)

## 주요 기능

- 엑셀 업로드 (갑지/을지)
- 자동 코드매핑 및 규칙 적용
- 유효성 검증 (필수값 → 형식 → 비즈니스)
- KPIS API 자동 제출 (MA111~MA114)
- 엑셀 내보내기 (수동 보고용)
