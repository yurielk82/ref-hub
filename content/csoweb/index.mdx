# CSO Web

건설안전관리 통합 플랫폼

## 개요

| 항목 | 내용 |
|------|------|
| **스택** | Next.js 14 (App Router) + TypeScript + Tailwind CSS |
| **DB** | Supabase (PostgreSQL + RLS) |
| **호스팅** | Vercel |
| **인증** | JWT + Supabase Auth |

## 주요 기능

- 현장 일일안전보고 관리
- 근로자/장비 출역 현황
- 안전점검 체크리스트
- 대시보드 통계
