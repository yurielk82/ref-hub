# 체크인 관리

셀러의 스튜디오 입장을 확인하고 체크인·체크아웃을 처리하는 절차를 안내합니다.

## 체크인 방법 3가지

| 방법 | 설명 | 권장 환경 |
|------|------|----------|
| **QR 스캔** | 셀러 앱의 QR 코드를 카메라로 스캔 | 모바일 앱 |
| **PIN 번호** | 셀러가 제시하는 6자리 숫자 입력 | 웹·모바일 |
| **수동 입력** | 예약번호(예: RES-20240101-001)를 직접 입력 | 웹·모바일 |

## 체크인 처리 — 웹

웹 대시보드에서는 PIN 번호와 수동 입력 방법으로 체크인을 처리할 수 있습니다.

1. 사이드바에서 **체크인 관리**를 클릭합니다
2. 체크인 방법(QR 코드 / PIN 번호 / 수동 입력)을 선택합니다
3. 해당 방법에 맞게 값을 입력합니다
4. **체크인 확인** 버튼을 클릭하거나 Enter 키를 누릅니다
5. 화면 상단에 성공 또는 실패 메시지가 표시됩니다

### QR 코드 (웹)

웹에서 QR 방법을 선택하면 외부 바코드 스캐너(USB 또는 블루투스 연결)를 사용할 수 있습니다. 스캐너가 연결된 경우 QR 코드를 스캔하면 입력 필드에 자동으로 값이 채워지며, Enter 키로 체크인이 처리됩니다.

카메라 내장 스캔은 **모바일 앱에서만** 지원됩니다.

### PIN 번호 (웹)

1. **PIN 번호** 탭을 선택합니다
2. 셀러가 앱에서 보여주는 6자리 PIN 번호를 입력합니다 (숫자만 입력 가능)
3. Enter 키 또는 **체크인 확인** 버튼을 클릭합니다

### 수동 입력 (웹)

1. **수동 입력** 탭을 선택합니다
2. 예약번호(예: `RES-20240101-001`)를 입력합니다
3. Enter 키 또는 **체크인 확인** 버튼을 클릭합니다

## 체크인 처리 — 모바일 앱

모바일 앱에서는 QR 카메라 스캔을 포함한 세 가지 방법 모두 사용할 수 있습니다.

1. 하단 탭에서 **운영** > **체크인**으로 이동합니다
2. 방법 카드(QR 스캔 / PIN 입력 / 수동)를 탭해 선택합니다

### QR 스캔 (모바일)

1. **QR 스캔** 카드를 탭합니다
2. 카메라 권한이 요청되면 **허용**을 탭합니다
3. 카메라 뷰파인더가 활성화되면 셀러 앱의 QR 코드를 화면에 비춥니다
4. QR 코드가 인식되면 자동으로 체크인이 처리됩니다 (추가 버튼 탭 불필요)
5. 인식 후 3초 동안은 재스캔이 차단되어 중복 처리를 방지합니다

카메라 권한을 이미 거부한 경우, 앱 설정 > 권한에서 카메라를 다시 허용해야 합니다.

### PIN 입력 (모바일)

1. **PIN 입력** 카드를 탭합니다
2. 숫자 키보드가 열리면 셀러가 제시하는 PIN 번호를 입력합니다
3. **체크인** 버튼을 탭합니다

### 수동 입력 (모바일)

1. **수동** 카드를 탭합니다
2. 예약번호 또는 예약 ID를 직접 입력합니다 (예: `SG-20260307-001`)
3. **체크인** 또는 **체크아웃** 버튼을 탭합니다

## 체크아웃 처리

셀러가 스튜디오 이용을 마치고 퇴실할 때 체크아웃을 처리합니다. 슬롯 종료 시간이 되면 상태가 자동으로 완료로 변경되지만, 조기 퇴실 시 수동으로 처리합니다.

### 웹에서 체크아웃

1. **체크인 관리** 화면 하단의 **최근 체크인 내역** 테이블을 확인합니다
2. 체크아웃할 셀러 행의 상태가 **체크인 중**인지 확인합니다
3. 해당 행 오른쪽의 **체크아웃** 버튼을 클릭합니다

### 모바일에서 체크아웃

수동 입력 또는 PIN 방법 선택 후 예약 정보를 입력한 다음 **체크아웃** 버튼을 탭합니다.

## 체크인 내역 확인

웹 화면 하단의 **최근 체크인 내역** 테이블에서 오늘 처리된 체크인 기록을 확인할 수 있습니다.

| 컬럼 | 설명 |
|------|------|
| 예약 번호 | 체크인에 사용된 예약 식별 번호 |
| 셀러명 | 체크인한 셀러 이름 |
| 스튜디오 | 이용한 스튜디오 이름 |
| 체크인 시간 | 체크인이 처리된 시각 |
| 체크아웃 시간 | 체크아웃 처리 시각 (미처리 시 `—` 표시) |
| 상태 | 체크인 중 / 체크아웃 |

## 오류 상황 대응

| 오류 메시지 | 원인 | 조치 |
|------------|------|------|
| 예약 번호를 입력해 주세요 | 입력값 없이 체크인 시도 | 예약번호·PIN·QR 중 하나를 입력 후 재시도 |
| 체크인 처리 중 오류가 발생했습니다 | 네트워크 오류 또는 유효하지 않은 예약 | 예약번호를 다시 확인하고 수동 입력으로 재시도 |
| 카메라 권한이 필요합니다 | 앱 카메라 권한 미허용 | 앱 설정에서 카메라 권한 허용 후 재시도 |
