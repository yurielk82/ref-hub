# 사용자 관리

회원 승인과 역할 관리 방법을 설명합니다.

## 역할 체계

| 역할 | 권한 |
|------|------|
| **관리자** | 전체 메뉴 접근, 설정 변경, 사용자 관리 |
| **뷰어** | 대시보드 조회, AI 설정만 접근 (프리셋·시스템 상태 숨김) |

## 회원 승인

1. 사이드바에서 **설정 > 사용자 관리**를 선택합니다
2. **대기 중** 목록에서 가입 신청자를 확인합니다
3. **승인** 또는 **거부** 버튼을 클릭합니다

> **주의**: 승인하기 전까지 사용자는 대기 화면만 보입니다.

## 역할 변경

1. **사용자 관리** 페이지에서 대상 사용자를 찾습니다
2. 역할 드롭다운에서 원하는 역할을 선택합니다
3. 변경 즉시 적용됩니다

## 인증 방식

- Supabase Auth 기반 이메일/비밀번호 인증
- 사내 시스템이므로 `authenticated` 역할에 전체 데이터 접근 허용
- 세분화된 권한은 애플리케이션 레벨에서 역할 기반으로 관리
