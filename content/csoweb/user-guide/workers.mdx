# 회원 관리

CSO 정산 포털의 회원 정보 확인, 수정, 회원가입 절차를 안내합니다. 일반회원(CSO 업체)은 자사 정보를 관리하고, 신규 업체는 회원가입을 신청할 수 있습니다.

## 내 정보 확인

로그인 후 `/profile` 메뉴에서 현재 로그인한 계정의 정보를 확인할 수 있습니다.

### 표시 항목

| 항목 | 설명 | 수정 가능 |
|------|------|----------|
| 사업자번호 | 로그인 ID로 사용되는 사업자등록번호 | 불가 |
| 회사명 | CSO 업체의 법인/상호명 | 불가 |
| 대표자명 | 사업자등록증 상의 대표자 | 불가 |
| 담당자명 | 정산 업무 담당자 이름 | 가능 |
| 연락처 | 담당자 전화번호 | 가능 |
| 이메일 | 정산서 수신 및 알림용 이메일 | 가능 |
| 가입일 | 회원가입 승인일 | 불가 |
| 상태 | 현재 계정 상태 (활성/비활성) | 불가 |

> 사업자번호, 회사명, 대표자명은 사업자등록증 기반 정보이므로 직접 수정할 수 없습니다. 변경이 필요한 경우 관리자(영업관리팀)에게 요청하세요.

### 정보 수정 방법

1. 좌측 메뉴에서 **내 정보**를 클릭합니다.
2. 수정 가능한 항목(담당자명, 연락처, 이메일)을 변경합니다.
3. **저장** 버튼을 클릭합니다.
4. 변경 사항이 즉시 반영됩니다.

## 비밀번호 변경

보안을 위해 주기적인 비밀번호 변경을 권장합니다.

### 변경 절차

1. `/profile` 화면에서 **비밀번호 변경** 영역을 찾습니다.
2. **현재 비밀번호**를 입력합니다.
3. **새 비밀번호**를 입력합니다.
4. **새 비밀번호 확인**에 동일한 비밀번호를 다시 입력합니다.
5. **변경** 버튼을 클릭합니다.

### 비밀번호 규칙

| 규칙 | 조건 |
|------|------|
| 최소 길이 | 8자 이상 |
| 구성 | 영문, 숫자, 특수문자 조합 권장 |
| 이전 비밀번호 | 현재 비밀번호와 동일 불가 |

## 회원가입 신청

CSO 정산 포털은 관리자 승인제로 운영됩니다. 신규 CSO 업체는 아래 절차를 따라 가입을 신청합니다.

### 가입 절차

```
신청서 작성 → 관리자 검토 → 승인/반려 → 로그인 가능
```

#### 1단계: 신청서 작성

로그인 화면에서 **회원가입** 링크를 클릭한 후 아래 정보를 입력합니다.

| 입력 항목 | 필수 | 설명 |
|----------|------|------|
| 사업자번호 | 필수 | 10자리 사업자등록번호 (하이픈 없이) |
| 회사명 | 필수 | 사업자등록증 상의 상호 |
| 대표자명 | 필수 | 사업자등록증 상의 대표자 |
| 담당자명 | 필수 | 정산 업무 실무 담당자 |
| 연락처 | 필수 | 담당자 전화번호 |
| 이메일 | 필수 | 정산서 수신 이메일 |
| 비밀번호 | 필수 | 로그인에 사용할 비밀번호 |

#### 2단계: 관리자 검토

신청서가 제출되면 관리자(영업관리팀)가 확인합니다. 검토에는 영업일 기준 1~2일이 소요될 수 있습니다.

관리자는 다음을 확인합니다.
- 사업자번호의 유효성
- CSO 계약 관계 확인
- 중복 가입 여부

#### 3단계: 승인 또는 반려

- **승인**: 계정이 활성화되고, 등록한 이메일로 승인 알림이 발송됩니다. 이후 사업자번호와 비밀번호로 로그인할 수 있습니다.
- **반려**: 반려 사유와 함께 알림이 발송됩니다. 사유를 확인한 후 재신청할 수 있습니다.

### 가입 후 CSO 매칭

회원가입이 승인되더라도 CSO 매칭이 완료되어야 정산 데이터를 조회할 수 있습니다. 매칭은 관리자가 엑셀 업체명과 회원 사업자번호를 연결하는 작업입니다. 매칭이 완료되면 정산 조회 화면에 자사 데이터가 표시됩니다.

> CSO 매칭에 대한 자세한 내용은 [CSO 매칭](/csoweb/user-guide/equipment) 페이지를 참조하세요.
