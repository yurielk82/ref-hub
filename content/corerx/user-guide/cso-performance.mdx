# CSO 성과

CSO(Contract Sales Organization, 위탁영업조직) 성과 탭은 외부 영업 조직의 처방 유도 성과를 분석합니다. 침투율 매트릭스, 흡수율(3M/6M/12M), 정산 현황을 종합적으로 확인할 수 있습니다.

## CSO 구조

CSO는 제약사가 외부 전문 영업조직에 위탁하여 의약품 처방을 유도하는 모델입니다.

| 구분 | 설명 |
|------|------|
| 제약사 | 의약품 제조·공급, CSO에 수수료 지급 |
| CSO 업체 | 외부 영업조직, 담당 거래처에 방문하여 처방 유도 |
| 담당 영업사원 | CSO 소속 개인 영업사원 |
| 거래처 | 병원/약국, 실제 처방 발생처 |

## 위젯 구성

| 위젯 | 유형 | 표시 내용 |
|------|------|-----------|
| 평균 흡수율 | KPI 카드 | 전체 CSO 평균 흡수율 (%) |
| CSO별 처방금액 | 가로 막대 차트 | CSO 업체별 처방유도 금액 (백만원) |
| 정산 현황 테이블 | 테이블 | CSO별 처방월, 처방금액, 수수료, 수수료율 |

## 침투율 매트릭스

침투율(선점율)은 CSO 업체가 담당하는 거래처-품목 조합의 비율입니다.

### 선점 개념

- **선점**: 특정 CSO 업체가 특정 거래처+품목 조합의 영업을 담당하는 것
- 동일 거래처+품목에 2개 이상의 CSO가 선점하면 **중복 경고**가 표시됩니다
- 선점은 자동 추출(정산 데이터 기반)과 수동 등록으로 관리됩니다

### 중복 경고

중복 선점이 발생하면 다음 문제가 생깁니다:

| 문제 | 영향 |
|------|------|
| 수수료 이중 지급 | 동일 처방에 대해 복수 CSO에 수수료가 발생 |
| 성과 귀속 분쟁 | 어떤 CSO가 처방을 유도했는지 불명확 |
| 흡수율 왜곡 | 분모(공급량)가 중복 집계되어 흡수율이 낮아짐 |

> **주의**: 중복 경고가 표시되면 영업관리자가 선점 매트릭스를 검토하여 한쪽 CSO의 선점을 해제해야 합니다.

## 흡수율

흡수율은 제약사가 공급한 의약품이 CSO를 통해 실제 처방으로 전환된 비율입니다.

### 계산 방식

- **분자**: CSO가 유도한 처방 조제 금액 (csoweb_settlements 기준)
- **분모**: 제약사가 해당 거래처에 공급한 금액 (kpis_gap_rows 기준)
- **흡수율(%)** = CSO 처방금액 / 갑지 공급금액 x 100

### 기간별 분석 (3M/6M/12M)

공급과 처방 사이에는 2~3개월의 시차가 있습니다. 이를 감안하여 기간별 누적 흡수율을 제공합니다.

| 기간 | 용도 | 특징 |
|------|------|------|
| 3개월 | 단기 추세 | 최근 변화 빠르게 감지, 계절성 영향 큼 |
| 6개월 | 중기 평가 | CSO 성과 중간 점검에 적합 |
| 12개월 | 연간 종합 | 가장 안정적 평가, 계절성 상쇄 |

### 흡수율 해석 기준

| 흡수율 범위 | 상태 | 의미 | 권장 조치 |
|------------|------|------|-----------|
| 100% 초과 | 과다처방 | 처방량이 공급량 초과 (타경로 공급 포함 가능) | 공급량 확인 |
| 80~100% | 정상 | 공급 대부분이 처방 전환됨 | 현행 유지 |
| 50~80% | 주의 | 상당량이 재고로 체류 | CSO 활동 점검 |
| 50% 미만 | 심각 | 절반 이상이 미전환 | 선점 해제/교체 검토 |
| 0% | 미활동 | 공급은 있으나 처방 없음 | CSO 계약 재검토 |

> **주의**: 공급월 M의 흡수율을 즉시 판단하면 과소평가됩니다. 최소 M+3 시점까지 대기 후 평가하세요. 3개월 누적 흡수율이 12개월 누적보다 낮은 것은 정상이며, 시차가 아직 반영되지 않은 것입니다.

## 정산 현황

CSO별 수수료 정산 상태를 테이블로 표시합니다.

### 정산 흐름

처방이 발생하면 다음 단계를 거쳐 정산됩니다:

1. 처방 발생 (환자 방문)
2. 처방 데이터 월별 집계
3. 정산 데이터 생성
4. 영업관리팀 확인 (pending → confirmed)
5. 재무팀 지급 (confirmed → paid)

### 수수료 구조

| 수수료 유형 | 설명 | 일반 범위 |
|------------|------|-----------|
| 제약수수료 | 제약사가 CSO 업체에 지급 | 처방금액의 5~15% |
| 담당수수료 | CSO가 담당 영업사원에 지급 | 처방금액의 3~10% |
| 인센티브 | 성과 기반 추가 수수료 | 처방금액의 0~5% |

### 정산 테이블 컬럼

| 컬럼 | 설명 |
|------|------|
| CSO 관리업체 | CSO 업체명 |
| 처방월 | 처방이 발생한 연월 |
| 처방금액 | 해당 월 처방조제액 합계 (백만원) |
| 제약수수료 합계 | 기본 수수료 + 인센티브 |
| 수수료율 | 제약수수료 합계 / 처방금액 x 100 (%) |
| 정산 상태 | pending / confirmed / paid / disputed |

### 활용 시나리오

1. **흡수율 50% 미만 CSO 발견**: 해당 CSO의 선점 거래처를 확인하고, 경쟁사 활동을 파악합니다.
2. **수수료율이 높은데 흡수율이 낮음**: 투자 효율이 떨어지므로, CSO 계약 조건 재검토를 고려합니다.
3. **특정 제품 흡수율 급락**: 경쟁제 출시, 급여 변경 등 외부 요인을 확인합니다.

> **주의**: CSO 역할로 로그인한 사용자는 본인 소속 CSO 업체의 데이터만 볼 수 있습니다. 타 CSO 업체의 성과 데이터는 표시되지 않습니다.
