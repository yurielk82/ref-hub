# 보고서 관리

정산서 데이터를 엑셀로 내보내는 방법과 표시 항목 설정이 내보내기에 미치는 영향을 안내합니다.

## 엑셀 내보내기

정산 조회 화면에서 현재 조회 중인 데이터를 엑셀 파일로 다운로드할 수 있습니다.

### 내보내기 방법

1. [정산 조회 화면](/csoweb/user-guide/dashboard)에서 원하는 **정산월**을 선택합니다.
2. 필요에 따라 검색어를 입력하여 데이터를 필터링합니다.
3. **표시 항목**에서 엑셀에 포함할 컬럼을 체크합니다.
4. 테이블 우측 상단의 **"엑셀 다운로드"** 버튼을 클릭합니다.
5. 브라우저의 다운로드 폴더에 엑셀 파일이 저장됩니다.

### 파일명 패턴

다운로드되는 엑셀 파일의 이름은 다음 형식으로 자동 생성됩니다.

| 항목 | 내용 |
|------|------|
| 파일명 형식 | `정산서_{정산월}.xlsx` |
| 파일명 예시 | `정산서_2026-01.xlsx` |
| 파일 형식 | `.xlsx` (Excel 2007 이상 호환) |

### 표시 항목과 내보내기의 관계

엑셀 내보내기는 화면에 표시된 컬럼과 동일한 컬럼만 포함합니다. 즉, 정산 조회 화면의 **표시 항목 체크박스**에서 선택한 컬럼만 엑셀 파일에 포함됩니다.

| 표시 항목 체크 상태 | 화면 표시 | 엑셀 포함 |
|:------------------:|:--------:|:--------:|
| 체크됨 | O | O |
| 해제됨 | X | X |
| 필수 항목 (해제 불가) | O | O |

> **주의**: 엑셀 다운로드 전에 필요한 컬럼이 모두 체크되어 있는지 반드시 확인하세요. 해제된 컬럼의 데이터는 엑셀에 포함되지 않습니다.

### 전체 컬럼 내보내기

47개 전체 컬럼을 모두 포함한 엑셀 파일이 필요한 경우, 표시 항목 체크박스에서 모든 컬럼을 선택한 후 다운로드하세요.

## 대량 데이터 내보내기 시 주의사항

정산 데이터가 많은 경우 엑셀 파일 생성에 시간이 걸릴 수 있습니다. 다음 사항에 유의하세요.

### 정상 동작

- **"엑셀 다운로드"** 버튼 클릭 후 파일이 생성될 때까지 잠시 기다려 주세요.
- 데이터 양에 따라 수 초에서 수십 초가 소요될 수 있습니다.
- 파일 생성이 완료되면 브라우저가 자동으로 다운로드를 시작합니다.

### 문제 발생 시 대처

| 증상 | 원인 | 해결 방법 |
|------|------|----------|
| 다운로드가 시작되지 않음 | 브라우저 팝업 차단 | 주소창 옆의 차단 아이콘 클릭 → "항상 허용" 선택 |
| 파일이 생성되지 않음 | 필터 조건에 맞는 데이터 없음 | 정산월, 검색어 등 필터 조건 확인 |
| 생성 시간이 너무 오래 걸림 | 데이터 양이 매우 많음 | 검색어로 범위를 좁힌 후 분할 다운로드 |

## 정산서 데이터 구조

정산서는 SIT(영업정보시스템)에서 추출한 엑셀 데이터를 기반으로 합니다. 엑셀 1행이 1건의 처방/거래에 해당하며, 최대 47개 컬럼으로 구성됩니다.

### 주요 컬럼

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| 처방월 | 처방이 발생한 연월 | 2026-01 |
| 정산월 | 정산 기준 연월 | 2026-02 |
| 거래처명 | 의약품 납품 대상 (병원, 약국 등) | OO약국 |
| 제품명 | 거래 대상 의약품 이름 | OO정 10mg |
| 수량 | 해당 건의 거래 수량 | 500 |
| 금액 | 해당 건의 거래 금액 | 2,500,000 |
| 제약_수수료 | CSO 업체에 지급되는 수수료 | 150,000 |

### 보조 컬럼

주요 컬럼 외에 정산 분석에 참고할 수 있는 컬럼입니다. 관리자가 표시 여부를 설정합니다.

| 분류 | 포함 컬럼 예시 |
|------|--------------|
| 거래처 정보 | 거래처 코드, 거래처 유형, 지역 |
| 제품 정보 | 제품 코드, 제형, 보험 코드, 단가 |
| 거래 정보 | 거래일자, 납품일자, 청구금액, 공급가 |
| CSO 정보 | CSO관리업체, 담당자명, 수수료율 |

## 월별 합계와 정산 조회의 활용

CSO 정산 포털에서는 정산 데이터를 두 가지 관점으로 제공합니다. 목적에 따라 적합한 화면을 선택하세요.

### 정산 조회 — 건별 상세 확인

- **용도**: 개별 거래 내역 확인, 특정 거래처/제품 검색
- **예시**: "OO약국에 납품된 XX정이 몇 건인지 확인", "특정 거래의 수수료를 확인"

### 월별 합계 — 기간별 추이 파악

- **용도**: 월별 수수료 추이 파악, 전월 대비 비교
- **예시**: "지난 6개월간 수수료 추이 확인", "전월 대비 수수료 변동 확인"

> 월별 합계의 총 수수료는 해당 월의 모든 개별 건 수수료를 합산한 값입니다. 정산 조회에서 동일 정산월을 선택한 후 제약수수료 합계 카드의 금액과 일치합니다.
