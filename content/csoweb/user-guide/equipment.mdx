# CSO 매칭

CSO 매칭은 정산 엑셀에 기재된 업체명과 포털에 가입된 회원의 사업자번호를 연결하는 작업입니다. 매칭이 완료되어야 일반회원이 자사 정산 데이터를 조회할 수 있습니다.

## CSO 매칭이란

CSO(위탁영업) 정산 프로세스에서 핵심적인 연결 단계입니다.

### 매칭이 필요한 이유

SIT(영업정보시스템)에서 추출한 정산 엑셀에는 **업체명**(텍스트)이 기재되어 있습니다. 반면 포털에 가입한 회원은 **사업자번호**로 식별됩니다. 이 두 정보를 연결해야 "이 정산 데이터는 이 회원의 것"이라는 관계가 성립합니다.

```
정산 엑셀                  포털 회원 DB
┌─────────────┐           ┌──────────────────┐
│ 업체명: ABC  │ ──매칭──▶ │ 사업자번호: 123-xx │
│ 수수료: 500만 │           │ 회사명: ABC영업    │
└─────────────┘           └──────────────────┘
```

### 매칭 흐름

1. 관리자가 SIT 엑셀을 포털에 업로드합니다.
2. 시스템이 엑셀의 업체명과 기존 매칭 테이블을 대조합니다.
3. 이미 매칭된 업체는 자동으로 연결됩니다.
4. 매칭되지 않은 업체는 관리자가 수동으로 연결합니다.

## 매칭 상태

각 CSO 업체의 매칭 상태는 다음 네 가지 중 하나입니다.

| 상태 | 코드 | 의미 | 정산 조회 |
|------|------|------|----------|
| 정상 | `normal` | 엑셀 업체명과 회원 사업자번호가 정상 연결됨 | 가능 |
| 미등록 | `unregistered` | 엑셀에 업체명이 있으나 포털에 해당 회원이 없음 | 불가 |
| 가입대기 | `pending_join` | 회원가입 신청은 했으나 아직 승인 전 | 불가 |
| 매칭없음 | `missing_match` | 회원은 존재하나 엑셀 업체명과의 연결이 없음 | 불가 |

### 상태별 대응 방법

#### 정상 (`normal`)

정상 상태에서는 별도 조치가 필요 없습니다. 정산 조회 화면에서 자사 데이터를 확인할 수 있습니다.

#### 미등록 (`unregistered`)

포털에 회원가입이 되어 있지 않은 상태입니다.

1. [회원가입 신청](/csoweb/user-guide/workers#회원가입-신청)을 진행합니다.
2. 관리자 승인을 기다립니다.
3. 승인 후 관리자가 CSO 매칭을 수행합니다.

#### 가입대기 (`pending_join`)

회원가입 신청서를 제출했으나 관리자 승인이 완료되지 않은 상태입니다.

- 승인까지 영업일 기준 1~2일이 소요됩니다.
- 급한 경우 관리자(영업관리팀)에게 직접 문의하세요.

#### 매칭없음 (`missing_match`)

포털에 회원으로 등록되어 있으나, 정산 엑셀의 업체명과 연결되지 않은 상태입니다. 주로 다음 경우에 발생합니다.

- 신규 가입 후 아직 매칭 작업이 진행되지 않음
- 엑셀에 기재된 업체명이 포털 등록 회사명과 다름 (예: "ABC" vs "ABC영업")
- 정산 엑셀에 해당 업체의 데이터가 아직 없음

## 매칭이 안 될 때

정산 조회 화면에 데이터가 표시되지 않는다면 CSO 매칭 상태를 확인해야 합니다.

### 확인 순서

1. **로그인 확인**: 정상적으로 로그인되어 있는지 확인합니다.
2. **처방월 확인**: 정산 조회 화면에서 올바른 처방월이 선택되어 있는지 확인합니다.
3. **매칭 상태 확인**: 관리자에게 현재 CSO 매칭 상태를 문의합니다.

### 관리자에게 매칭 요청

매칭이 필요한 경우 아래 정보를 관리자(영업관리팀)에게 전달합니다.

| 전달 항목 | 예시 |
|----------|------|
| 사업자번호 | 123-45-67890 |
| 회사명 | ABC영업 |
| 정산 엑셀 상의 업체명 | ABC (엑셀에 기재된 이름과 다를 수 있음) |
| 조회하려는 처방월 | 2026-01 |

관리자가 매칭을 완료하면 정산 조회 화면에 데이터가 즉시 표시됩니다. 별도의 로그아웃/로그인은 필요하지 않습니다.

## 매칭과 무결성 검사

관리자는 `/admin/integrity` 화면에서 CSO 매칭의 무결성을 검사합니다. 무결성 검사는 다음을 확인합니다.

- 엑셀에 존재하지만 매칭되지 않은 업체 목록
- 매칭은 되어 있으나 해당 처방월 데이터가 없는 회원 목록
- 동일 업체가 여러 사업자번호에 중복 매칭된 경우

일반회원은 무결성 검사를 직접 수행할 수 없으며, 매칭 관련 문제가 있으면 관리자에게 문의합니다.
