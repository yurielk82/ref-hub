# 시작하기

CSO Web 사용을 위한 초기 설정 가이드입니다.

## 회원가입

1. CSO Web 접속 후 **회원가입** 버튼을 클릭합니다
2. 사업자번호, 회사명, 담당자 이름, 이메일, 비밀번호를 입력합니다
3. 가입 신청이 완료되면 관리자(영업관리팀)에게 승인 요청 이메일이 자동 발송됩니다
4. 관리자가 승인하면 로그인이 가능합니다

## 로그인

1. 브라우저에서 CSO Web 주소에 접속합니다
2. **사업자번호**와 **비밀번호**를 입력하여 로그인합니다
3. 최초 로그인 시 비밀번호 변경을 권장합니다

## 비밀번호 재설정

비밀번호를 잊은 경우:
1. 로그인 화면에서 **비밀번호 찾기**를 클릭합니다
2. 등록된 이메일 주소를 입력합니다
3. 이메일로 발송된 재설정 링크를 클릭하여 새 비밀번호를 설정합니다

## 화면 구성

로그인 후 홈 대시보드가 표시됩니다. 상단 메뉴에서 각 기능에 접근할 수 있습니다.

| 메뉴 | 설명 |
|------|------|
| **홈** | 최근 정산 현황 요약 |
| **정산 조회** | 정산서 데이터 검색/필터/다운로드 |
| **월별 합계** | 월별 수수료 합산 통계 |
| **내 정보** | 프로필 및 비밀번호 변경 |
