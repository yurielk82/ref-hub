# 예약 관리

셀러(MEMBER)의 예약 요청을 검토하고 승인하거나 거절하는 절차를 안내합니다.

## 예약 상태 종류

| 상태 | 의미 |
|------|------|
| **승인 대기** | 셀러가 예약을 신청했으나 아직 검토 전 |
| **승인됨** | 운영자가 예약을 승인한 상태 |
| **거절** | 운영자가 예약을 거절한 상태 |
| **취소** | 셀러가 직접 예약을 취소한 상태 |
| **완료** | 이용이 종료된 상태 |
| **노쇼** | 체크인 없이 슬롯이 종료된 상태 |

## 예약 목록 조회

### 웹

상단 사이드바에서 **예약 관리**를 클릭합니다. 전체 예약 목록이 테이블 형태로 표시됩니다.

- 상단 탭(전체 / 승인 대기 / 승인됨 / 거절 / 완료 / 취소)을 클릭해 상태별로 필터링합니다
- 테이블 컬럼: 예약번호, 예약자, 스튜디오, 날짜, 시간, 상태, 작업
- 한 번에 최대 50건이 표시됩니다

### 모바일

하단 탭에서 **예약 관리**를 탭합니다.

- 상단 칩(대기 / 승인 / 완료 / 전체)으로 상태를 전환합니다
- 기본값은 **대기** 탭으로, 즉시 처리가 필요한 예약이 먼저 보입니다

## 예약 승인

### 단건 승인

#### 웹

1. 예약 목록에서 승인할 예약 행을 확인합니다
2. 해당 행의 **승인** 버튼을 클릭합니다
3. 승인이 완료되면 상태 배지가 **승인됨**으로 즉시 변경됩니다

#### 모바일

1. **예약 관리** 화면에서 대기 탭을 확인합니다
2. 처리할 예약 카드의 **승인** 버튼을 탭합니다
3. 로딩 인디케이터가 사라지면 승인 완료입니다

### 일괄 승인 (웹 전용)

승인 대기 예약이 여러 건인 경우 체크박스로 선택 후 한 번에 처리할 수 있습니다.

1. **승인 대기** 탭으로 이동합니다
2. 처리할 예약 행 왼쪽의 체크박스를 클릭합니다
   - 헤더의 체크박스를 클릭하면 현재 페이지의 모든 승인 대기 예약이 선택됩니다
3. 선택된 건수가 표시된 **선택 승인 (N건)** 버튼이 우상단에 나타납니다
4. 버튼을 클릭하면 선택된 예약이 모두 승인됩니다

모바일 대시보드의 **승인 대기 예약** 섹션에서도 개별 승인 버튼으로 빠르게 처리할 수 있습니다.

## 예약 거절

거절 시에는 반드시 사유를 입력해야 합니다. 입력한 사유는 셀러에게 카카오 알림톡으로 전달됩니다.

### 웹

1. 예약 목록에서 거절할 예약 행의 **거절** 버튼을 클릭합니다
2. **거절 사유 입력** 다이얼로그가 열립니다
3. 거절 사유를 입력합니다 (최대 500자, 필수)
4. **거절 확인** 버튼을 클릭합니다
5. 상태 배지가 **거절**로 변경됩니다

### 모바일

1. 예약 카드의 **거절** 버튼을 탭합니다
2. 화면 상단에 거절 사유 입력 카드가 펼쳐집니다
3. 사유를 입력한 후 **거절** 버튼을 탭합니다
4. 취소하려면 **취소** 버튼을 탭합니다

## 승인/거절 기준

| 고려 사항 | 확인 방법 |
|----------|----------|
| 슬롯 충돌 여부 | 같은 날짜·시간대에 이미 승인된 예약이 있는지 확인 |
| 셀러 이용 이력 | 과거 노쇼·시설 훼손 이력을 회원 관리에서 조회 |
| 이용 목적 적합성 | 스튜디오 용도(의류 라이브커머스)에 부합하는지 판단 |

## 승인 후 안내

예약이 승인되면 셀러에게 카카오 알림톡이 자동으로 발송됩니다. 알림톡에는 스튜디오명, 날짜, 시간, 체크인 방법(QR 코드 또는 PIN)이 포함됩니다.
