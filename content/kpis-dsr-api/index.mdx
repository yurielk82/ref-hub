# KPIS DSR API

일일안전보고서(DSR) API 시스템

## 개요

| 항목 | 내용 |
|------|------|
| **스택** | Express 5 + React 19 (Vite) + Radix UI |
| **DB** | SQLite |
| **호스팅** | Vercel (프론트) + NHN PM2 (API) |
| **인증** | Google OAuth |

## 주요 기능

- 일일안전보고서(DSR) CRUD
- Google OAuth 기반 인증
- PDF 보고서 생성/다운로드
- 실시간 데이터 조회 API
