# EV Motor Reliability

전기차 모터 신뢰성 분석 시스템

## 개요

| 항목 | 내용 |
|------|------|
| **스택** | Python FastAPI (API) + Next.js 15 (Web) |
| **DB** | Neon PostgreSQL (선택) |
| **호스팅** | 자체 (Uvicorn) |
| **인증** | 미적용 |

## 주요 기능

- 모터 신뢰성 데이터 분석
- Weibull 분석 및 시각화
- 고장 예측 모델
- 분석 결과 대시보드
