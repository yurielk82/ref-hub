# 보고서 관리

정산서 데이터의 구조를 이해하고, 필터링과 엑셀 내보내기를 활용하여 필요한 정산 보고서를 생성하는 방법을 안내합니다.

## 정산서 데이터 구조

정산서는 SIT(영업정보시스템)에서 추출한 엑셀 데이터를 기반으로 합니다. 엑셀 1행이 1건의 처방/거래에 해당하며, 최대 47개 컬럼으로 구성됩니다.

### 주요 컬럼 설명

정산 데이터에서 가장 자주 사용하는 핵심 컬럼입니다.

| 컬럼명 | 설명 | 예시 |
|--------|------|------|
| 처방월 | 처방이 발생한 연월 | 2026-01 |
| 거래처명 | 의약품 납품 대상 (병원, 약국 등) | OO병원, XX약국 |
| 제품명 | 거래된 의약품 이름 | OO정 10mg |
| 수량 | 해당 건의 거래 수량 | 500 |
| 수수료 | CSO 업체에 지급되는 수수료 금액 | 150,000 |

### 보조 컬럼

주요 컬럼 외에 정산 분석에 참고할 수 있는 컬럼입니다. 관리자가 `/admin/columns`에서 표시 여부를 설정합니다.

| 분류 | 포함 컬럼 예시 |
|------|--------------|
| 거래처 정보 | 거래처 코드, 거래처 유형, 지역, 담당 영업사원 |
| 제품 정보 | 제품 코드, 제형, 보험 코드, 단가 |
| 거래 정보 | 거래일자, 납품일자, 청구금액, 공급가 |
| CSO 정보 | CSO 업체명, 담당자명, 수수료율 |

> 47개 컬럼 전체 목록은 관리자에게 문의하거나 엑셀 다운로드 후 확인할 수 있습니다.

## 데이터 필터링

정산 조회 화면(`/dashboard`)에서 다양한 조건으로 데이터를 필터링할 수 있습니다.

### 단일 필터

하나의 조건으로 데이터를 걸러냅니다.

1. **처방월 필터**: 조회할 처방월을 선택합니다. 기본값은 가장 최근 처방월입니다.
2. **거래처 필터**: 특정 거래처의 정산 내역만 표시합니다.
3. **제품 필터**: 특정 제품의 정산 내역만 표시합니다.

### 복합 필터

여러 필터를 동시에 적용하여 데이터를 더 좁힐 수 있습니다.

예시: "2026년 1월 + OO병원 + XX정" → 2026년 1월에 OO병원에 납품된 XX정의 정산 내역만 표시

### 검색

필터 외에 텍스트 검색으로 데이터를 찾을 수 있습니다. 검색창에 키워드를 입력하면 거래처명, 제품명 등 텍스트 컬럼에서 부분 일치하는 행이 표시됩니다.

## 엑셀 내보내기

현재 조회 중인 정산 데이터를 엑셀 파일(`.xlsx`)로 내보낼 수 있습니다.

### 내보내기 방법

1. 정산 조회 화면(`/dashboard`)에서 필요한 필터 조건을 설정합니다.
2. 테이블 우측 상단의 **엑셀 다운로드** 버튼을 클릭합니다.
3. 브라우저의 다운로드 폴더에 `.xlsx` 파일이 저장됩니다.

### 내보내기 파일 구성

| 항목 | 설명 |
|------|------|
| 파일 형식 | `.xlsx` (Excel 2007 이상) |
| 포함 데이터 | 현재 필터 조건에 맞는 행만 포함 |
| 포함 컬럼 | 현재 테이블에 표시된 컬럼 |
| 서식 | 숫자 컬럼에 천 단위 구분자 적용 |

### 내보내기가 안 될 때

- 브라우저의 **팝업 차단**이 활성화되어 있으면 다운로드가 차단될 수 있습니다. 팝업 허용 후 다시 시도하세요.
- 데이터가 너무 많으면 처리 시간이 걸릴 수 있습니다. 필터를 적용하여 데이터 범위를 줄여보세요.

## 월별 합계와 정산 데이터 차이

CSO 정산 포털에서는 정산 데이터를 두 가지 관점으로 제공합니다.

### 정산 조회 (`/dashboard`)

- **단위**: 건별 (1행 = 1건의 처방/거래)
- **용도**: 개별 거래 내역 확인, 특정 거래처/제품 검색
- **컬럼**: 최대 47개 상세 컬럼
- **적합한 상황**: "OO병원에 납품된 XX정이 몇 건인지 확인", "특정 거래의 수수료를 확인"

### 월별 합계 (`/monthly-summary`)

- **단위**: 월별 (1행 = 1개월 합산)
- **용도**: 월별 수수료 추이 파악, 기간별 비교
- **컬럼**: 처방월, 총 건수, 총 수수료
- **적합한 상황**: "지난 6개월간 수수료 추이 확인", "전월 대비 수수료 변동 확인"

> 월별 합계의 총 수수료는 해당 월의 모든 개별 건 수수료를 합산한 값입니다. 정산 조회에서 동일 처방월 필터를 적용한 후 수수료 합계와 일치합니다.
