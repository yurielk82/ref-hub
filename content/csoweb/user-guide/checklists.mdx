# 자주 묻는 질문

CSO 정산 포털 사용 중 자주 발생하는 문제와 해결 방법을 안내합니다.

## 로그인이 안 될 때

로그인에 실패하는 경우 아래 원인을 순서대로 확인하세요.

### 원인 1: 계정 미승인

회원가입 신청 후 관리자 승인이 완료되지 않은 상태에서는 로그인할 수 없습니다.

- 가입 신청 후 영업일 기준 1~2일 후에 다시 시도하세요.
- 승인이 완료되면 등록한 이메일로 승인 완료 안내 메일이 발송됩니다.
- 급한 경우 관리자(영업관리팀)에게 승인 상태를 문의하세요.

### 원인 2: 사업자번호 오입력

사업자번호 입력 시 하이픈은 시스템이 자동으로 입력합니다. 숫자 10자리만 정확히 입력하세요.

| 입력 방식 | 예시 | 결과 |
|----------|------|------|
| 올바른 입력 | `1234567890` | 시스템이 `123-45-67890`으로 자동 변환, 로그인 성공 |
| 하이픈 직접 입력 | `123-45-67890` | 로그인 실패 가능 |
| 공백 포함 | `123 456 7890` | 로그인 실패 |

### 원인 3: 비밀번호 오류

비밀번호는 대소문자를 구분합니다. 키보드의 Caps Lock이 켜져 있지 않은지 확인하세요. 비밀번호를 분실한 경우 아래 [비밀번호를 잊었을 때](#비밀번호를-잊었을-때) 항목을 참조하세요.

---

## 정산 데이터가 보이지 않을 때

정산 조회 화면에 데이터가 표시되지 않는 경우 아래 원인을 확인하세요.

### 원인 1: CSO 매칭 미완료

가장 흔한 원인입니다. 회원가입이 승인되었더라도 CSO 매칭이 완료되어야 자사의 정산 데이터가 표시됩니다.

- 관리자에게 사업자번호와 회사명을 전달하여 매칭 상태를 확인하세요.
- 매칭에 대한 자세한 내용은 [CSO 매칭](/csoweb/user-guide/equipment) 페이지를 참조하세요.

### 원인 2: 정산월 선택 오류

정산 조회 화면 상단의 드롭다운에서 정산월이 올바르게 선택되어 있는지 확인하세요. 다른 정산월을 선택하여 데이터가 있는지 확인해 보세요.

### 원인 3: 데이터 미업로드

관리자가 해당 정산월의 엑셀 데이터를 아직 업로드하지 않았을 수 있습니다. 관리자에게 해당 월의 데이터 업로드 일정을 문의하세요.

---

## 비밀번호를 잊었을 때

비밀번호를 분실한 경우 이메일을 통해 재설정할 수 있습니다.

### 이메일로 재설정하는 방법

1. 로그인 화면에서 **"비밀번호를 잊으셨나요?"** 링크를 클릭합니다.
2. 가입 시 등록한 **사업자번호**와 **이메일**을 입력합니다.
3. **"재설정 링크 발송"** 버튼을 클릭합니다.
4. 이메일로 수신한 **"비밀번호 재설정하기"** 버튼을 클릭합니다.
5. 새 비밀번호를 입력하고 **"비밀번호 변경"** 버튼을 클릭합니다.

> 재설정 링크는 **30분** 동안만 유효하며, **1회만** 사용할 수 있습니다. 만료된 경우 위 절차를 처음부터 다시 진행하세요.

### 이메일도 기억나지 않을 때

재설정 이메일을 받을 수 없는 상황이라면 관리자(영업관리팀)에게 직접 연락하여 비밀번호 초기화를 요청하세요.

1. 관리자가 비밀번호를 초기화하면 임시 비밀번호가 발급됩니다.
2. 임시 비밀번호로 로그인합니다.
3. 최초 로그인 시 자동으로 비밀번호 변경 화면이 표시됩니다.
4. 새 비밀번호를 설정하면 정상적으로 사용할 수 있습니다.

### 재설정 이메일이 도착하지 않을 때

- 스팸/정크 메일 폴더를 확인하세요.
- 가입 시 등록한 이메일 주소가 정확한지 확인하세요.
- 그래도 도착하지 않으면 관리자에게 등록된 이메일 주소 확인을 요청하세요.

## 사업자번호 변경이 필요할 때

사업자번호는 CSO 정산 포털 계정의 고유 식별자로, 한 번 등록하면 변경할 수 없습니다. 사업자번호가 변경된 경우(법인 전환, 사업자 신규 발급 등) 아래 절차를 따르세요.

### 처리 절차

1. **새 사업자번호로 신규 회원가입**을 신청합니다.
2. 관리자에게 연락하여 **기존 계정 삭제**를 요청합니다.
3. 관리자가 새 계정을 승인하고 CSO 매칭을 다시 설정합니다.

> 기존 계정의 과거 정산 데이터 이력이 필요한 경우, 관리자에게 미리 알려주세요. 데이터 이관 여부를 확인받을 수 있습니다.

## 업체 정보 수정 반영 시점

[내 정보](/csoweb/user-guide/workers) 화면에서 업체명, 연락처, 이메일 등을 수정한 경우, 반영 시점에 대해 안내합니다.

### 즉시 반영되는 항목

| 항목 | 설명 |
|------|------|
| 프로필 화면 | 수정한 값이 화면에 바로 표시됩니다 |
| 로그인 세션 | 상단 메뉴의 업체명 등이 즉시 갱신됩니다 |
| 이메일 알림 | 변경된 이메일로 다음 알림부터 발송됩니다 |

### 반영되지 않는 항목

| 항목 | 설명 |
|------|------|
| 기존 정산서 데이터 | 이미 업로드된 과거 데이터에는 반영되지 않습니다 |

정산서 데이터는 **업로드 시점의 정보**를 기준으로 생성됩니다. 프로필에서 업체명을 수정하더라도 이미 업로드된 과거 정산 데이터의 업체명은 변경되지 않습니다. 수정된 정보는 이후 관리자가 새로 업로드하는 정산 데이터부터 반영됩니다.

## 엑셀 내보내기가 안 될 때

정산 조회 화면에서 **"엑셀 다운로드"** 버튼을 클릭해도 파일이 다운로드되지 않는 경우 아래 원인을 확인하세요.

### 원인 1: 브라우저 팝업 차단

일부 브라우저에서 파일 다운로드를 팝업으로 인식하여 차단할 수 있습니다.

| 브라우저 | 차단 해제 방법 |
|---------|--------------|
| Chrome | 주소창 우측의 차단 아이콘 클릭 → **"항상 허용"** 선택 |
| Edge | 주소창 우측의 차단 아이콘 클릭 → **"허용"** 선택 |
| Safari | 환경설정 → 웹사이트 → 팝업 창 → 해당 사이트 **"허용"** |

### 원인 2: 데이터 없음

현재 필터 조건에 맞는 데이터가 없으면 다운로드할 파일이 생성되지 않습니다.

- 정산월이 올바르게 선택되어 있는지 확인하세요.
- 검색어가 입력되어 있다면 지우고 다시 시도하세요.

### 원인 3: 대용량 데이터

데이터가 매우 많은 경우 파일 생성에 시간이 걸릴 수 있습니다.

- **"엑셀 다운로드"** 버튼 클릭 후 잠시 기다려 주세요.
- 반복적으로 실패하면 검색어를 입력하여 데이터 범위를 줄인 후 분할 다운로드를 시도하세요.

### 원인 4: 표시 항목 미선택

엑셀에 포함할 컬럼이 하나도 선택되지 않은 경우 파일이 정상적으로 생성되지 않을 수 있습니다. 표시 항목 체크박스에서 필요한 컬럼을 선택한 후 다시 시도하세요.

## 기타 문의

위 내용으로 해결되지 않는 문제는 관리자(영업관리팀)에게 문의하세요. 문의 시 아래 정보를 함께 전달하면 빠른 처리가 가능합니다.

| 전달 항목 | 설명 |
|----------|------|
| 사업자번호 | 로그인에 사용하는 10자리 번호 |
| 화면 캡처 | 문제가 발생한 화면의 스크린샷 |
| 발생 시점 | 문제가 발생한 날짜와 시간 |
| 브라우저 정보 | 사용 중인 브라우저 이름과 버전 (Chrome, Edge 등) |

> 권장 브라우저는 **Chrome** 또는 **Edge** 최신 버전입니다. 화면이 깨져 보이는 경우 브라우저를 최신 버전으로 업데이트하고, 캐시를 삭제한 후 새로고침(`Ctrl + Shift + R`)하세요.
