# 시작하기

StudioGo 앱 사용을 위한 초기 설정 가이드입니다.

## 앱 설치

1. App Store(iOS) 또는 Google Play(Android)에서 **StudioGo**를 검색합니다
2. 앱을 설치하고 실행합니다

## 회원가입/로그인

1. **카카오 로그인** 버튼을 탭합니다
2. 카카오 계정으로 인증을 완료합니다
3. 최초 로그인 시 프로필 정보(닉네임, 연락처)를 입력합니다

## 사용자 유형

| 유형 | 역할 | 권한 |
|------|------|------|
| **BJ(방송인)** | 스튜디오를 빌려 라이브커머스 방송 | 예약, 캘린더 조회 |
| **창고 운영자** | 예약 승인, 체크인 관리 | 예약 승인/거절, 체크인 처리 |
| **관리자** | 시스템 전체 관리 | 모든 기능 접근 |

## 핵심 용어

| 용어 | 설명 |
|------|------|
| **스튜디오** | 한정 자원으로 예약제 운영되는 방송 공간 |
| **슬롯** | 운영시간을 일정 간격으로 나눈 예약 단위 |
| **Hold** | 예약 위자드 중 2분간 슬롯을 임시 점유하는 상태 |
| **알림톡** | 카카오 비즈메시지 기반 알림 |
