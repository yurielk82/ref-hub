# 회원 조회

가입한 셀러 회원을 검색하고 상태를 확인하는 방법을 안내합니다.

운영자는 회원 목록을 조회하고 검색할 수 있습니다. 회원 정지·복구 등 상태 변경은 관리자(ADMIN) 권한이 필요합니다.

## 회원 상태 종류

| 상태 | 의미 |
|------|------|
| **활성** | 정상적으로 서비스를 이용할 수 있는 회원 |
| **대기** | 가입 후 관리자의 승인을 기다리는 회원 |
| **정지** | 이용이 일시 중단된 회원 |

## 회원 목록 조회 — 웹

1. 사이드바에서 **회원 관리**를 클릭합니다
2. 전체 회원 목록이 카드 형태로 표시됩니다

### 정보 표시

각 회원 행에는 다음 정보가 표시됩니다.

| 항목 | 설명 |
|------|------|
| 프로필 | 이름 + 전화번호 (아바타 포함) |
| 닉네임 | 플랫폼에서 사용하는 표시 이름 |
| 역할 | 셀러 / 운영자 / 관리자 |
| 상태 | 활성 / 대기 / 정지 |
| 가입일 | 최근 로그인 기준 날짜 |

**상세보기** 버튼을 클릭하면 해당 회원의 상세 페이지로 이동합니다.

### 검색

1. 상단 검색 입력 필드에 이름 또는 닉네임을 입력합니다
2. **검색** 버튼을 클릭하거나 Enter 키를 누릅니다
3. 입력한 키워드와 일치하는 회원만 표시됩니다
4. 검색어를 지우고 다시 검색하면 전체 목록으로 돌아옵니다

### 상태 필터

목록 우상단의 상태 버튼(전체 / 활성 / 대기 / 정지)을 클릭해 특정 상태의 회원만 볼 수 있습니다.

검색과 상태 필터를 동시에 적용할 수 있습니다. 예를 들어 닉네임으로 검색하면서 활성 상태로 필터링할 수 있습니다.

### 페이지네이션

한 페이지에 10명씩 표시됩니다. 하단의 **이전** / **다음** 버튼으로 페이지를 이동하며, 현재 페이지와 전체 페이지 수가 표시됩니다.

## 회원 목록 조회 — 모바일

1. 하단 탭에서 **운영** > **회원 관리**로 이동합니다
2. 상단 검색 필드에 이름, 닉네임, 전화번호를 입력해 검색합니다
3. 상태 칩(전체 / 활성 / 대기 / 정지)으로 필터링합니다

각 카드에는 닉네임, 등급(브론즈·실버·골드·플래티넘·다이아몬드), 이름, 전화번호, 누적 방송 횟수가 표시됩니다.

## 회원 등급(티어)

셀러는 누적 방송 실적에 따라 등급이 부여됩니다.

| 등급 | 설명 |
|------|------|
| **브론즈** | 초기 등급 |
| **실버** | 일정 방송 횟수 달성 |
| **골드** | 우수 셀러 |
| **플래티넘** | 고급 셀러 |
| **다이아몬드** | 최상위 등급 |

등급 기준 변경은 관리자만 가능합니다.

## 유의 사항

- 회원 정지·복구는 관리자(ADMIN) 권한이 필요합니다. 해당 조치가 필요한 경우 관리자에게 요청하세요.
- 대기 상태 회원은 관리자의 승인 전까지 예약을 신청할 수 없습니다.
- 검색 결과가 없을 때는 검색어를 줄이거나 상태 필터를 전체로 변경해 다시 시도하세요.
