PharmKPI

제약 KPI 대시보드 (v1.16)

매출·수금·마진·흡수율 등 제약 산업 핵심 지표를 실시간으로 시각화하는 대시보드. Oracle ERP 데이터를 자동 수집하고, Claude·Gemini·GPT 멀티 AI로 매출 추이를 분석합니다.

Dashboard Preview

Tech Stack

Next.js 16 Prisma 7 Supabase Anthropic SDK Recharts

주요 기능

01

실시간 매출·수금·마진 KPI 대시보드

전국 약국 및 병원별 매출 데이터를 실시간으로 집계하여 기업의 유동성과 수익성을 한눈에 파악할 수 있는 고성능 대시보드 UI를 제공합니다.

02

멀티 AI (Claude, Gemini, GPT) 매출 분석

단일 AI의 편향성을 극복하기 위해 Anthropic, Google, OpenAI의 엔진을 동시에 활용하여 시장 추세를 다각도로 분석하고 예측 리포트를 생성합니다.

03

PDF/Excel 리포트 자동 생성

매주 월요일 아침, 지난주 KPI 지표를 요약한 리포트를 PDF와 Excel 포맷으로 자동 생성하여 주요 이해관계자들에게 발송합니다.

04

Oracle ERP → Supabase ETL 파이프라인

온프레미스 Oracle ERP 환경의 데이터를 클라우드 기반 Supabase로 안전하고 효율적으로 동기화하는 ETL 파이프라인을 구축했습니다.

Oracle ERP 연동 + 멀티 AI 분석 대시보드

2024

전통적인 제약 산업 인프라와 최신 생성형 AI 기술을 결합하여, 단순 데이터 조회를 넘어 비즈니스 인사이트를 도출하는 차세대 솔루션입니다.
