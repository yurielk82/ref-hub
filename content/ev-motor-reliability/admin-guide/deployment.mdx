# 배포

EV Motor Reliability 배포 가이드입니다.

## 시스템 구성

| 컴포넌트 | 설명 | 실행 환경 |
|---------|------|----------|
| Python 분석 서버 | 시뮬레이션/분석 파이프라인 | 로컬 또는 서버 |
| C 펌웨어 | DSP 모터 제어 | TMS320F2838x 보드 |
| 자동화 스크립트 | 시리얼/JTAG 통신 | 시험 장비 연결 PC |

## Python 환경 설정

```bash
# 가상환경 생성 및 활성화
python3 -m venv venv
source venv/bin/activate

# 의존성 설치
pip install -r requirements.txt

# 시뮬레이션 서버 실행
python simulation/runner.py

# 자동화 파이프라인 실행
python automation/pipeline.py
```

## 펌웨어 빌드 및 플래시

1. **Code Composer Studio** 실행
2. 프로젝트 import: `firmware/` 디렉토리
3. 빌드: Project → Build Project
4. JTAG 디버거(SDS200i) 연결 확인
5. 플래시: Run → Debug (또는 UniFlash 사용)

## 설정 파일

`config.yaml`에 장비 연결 정보를 설정합니다 (Git 미추적):

```yaml
serial:
  port: /dev/ttyUSB0    # 시리얼 포트
  baudrate: 115200       # 보드레이트

jtag:
  type: SDS200i
  # 장비 IP는 보안상 기록하지 않음

analysis:
  output_dir: ./results
  plot_format: png
```

## 보안 주의사항

- 소속 기관명, 연구기관명을 코드/문서/커밋에 포함하지 않습니다
- 장비 IP, 네트워크 정보는 `config.yaml`에만 작성 (`.gitignore` 등록)
- `data/`와 `ref/`는 Git 추적 대상에서 제외합니다
