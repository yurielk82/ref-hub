# 캘린더

StudioGo 캘린더 기능을 활용하여 스튜디오 예약 현황을 한눈에 파악하는 방법을 안내합니다.

## 캘린더 화면 진입

하단 탭의 **캘린더** 아이콘을 탭하면 캘린더 화면으로 이동합니다. 기본적으로 현재 월이 표시되며, 내 예약이 있는 날짜에 마커가 표시됩니다.

## 뷰 모드

캘린더는 세 가지 뷰 모드를 제공합니다. 화면 상단의 뷰 전환 버튼으로 변경할 수 있습니다.

| 뷰 모드 | 설명 | 용도 |
|---------|------|------|
| **월간** | 한 달 전체를 달력 형태로 표시 | 월 단위 일정 파악 |
| **주간** | 한 주를 시간대별로 표시 | 주간 스케줄 관리 |
| **일간** | 하루를 시간대별로 상세 표시 | 특정 날짜의 슬롯 확인 |

### 월간 뷰

- 각 날짜 셀에 예약 건수와 상태별 색상 마커가 표시됩니다
- 날짜를 탭하면 해당 날짜의 일간 뷰로 이동합니다
- 좌우 스와이프로 이전/다음 월로 이동합니다
- 오늘 날짜는 강조 표시됩니다

### 주간 뷰

- 가로축이 요일(월~일), 세로축이 시간대로 구성됩니다
- 예약 블록이 시간대에 맞춰 배치됩니다
- 블록을 탭하면 해당 예약의 상세 정보를 확인할 수 있습니다
- 좌우 스와이프로 이전/다음 주로 이동합니다

### 일간 뷰

- 세로축이 시간대로 구성되며, 각 슬롯의 상태가 상세히 표시됩니다
- 빈 슬롯을 탭하면 바로 예약 화면으로 이동합니다
- 특정 스튜디오를 선택하면 해당 스튜디오의 슬롯만 표시됩니다

## 날짜별 예약 현황

### 날짜 선택

캘린더에서 날짜를 탭하면 해당 날짜의 예약 현황이 하단에 목록으로 표시됩니다.

### 현황 정보

각 예약 항목에는 다음 정보가 표시됩니다.

| 항목 | 설명 |
|------|------|
| **스튜디오명** | 예약한 스튜디오 이름 |
| **시간대** | 시작 시간 ~ 종료 시간 |
| **상태** | 대기/승인/체크인/완료 등 |
| **슬롯 수** | 연속 예약한 슬롯 개수 |

### 빈 날짜

예약이 없는 날짜를 선택하면 "예약 내역이 없습니다" 메시지와 함께 **스튜디오 검색** 바로가기 버튼이 표시됩니다.

## 슬롯 상태 표시

캘린더에서 슬롯은 색상으로 상태를 구분합니다.

| 색상 | 상태 | 설명 |
|------|------|------|
| **파란색** | 가용 | 예약 가능한 빈 슬롯 |
| **주황색** | Hold | 다른 사용자가 임시점유 중 (최대 2분) |
| **초록색** | 내 예약 (승인) | 내가 예약하고 승인된 슬롯 |
| **노란색** | 내 예약 (대기) | 내가 예약했으나 아직 승인 대기 중인 슬롯 |
| **회색** | 예약됨 | 다른 사용자가 예약한 슬롯 |
| **빗금** | 운영 외 | 운영 시간 밖이거나 차단된 슬롯 |

### 실시간 갱신

슬롯 상태는 **실시간으로 갱신**됩니다. 다른 사용자가 Hold를 걸거나 예약을 확정하면 캘린더에 즉시 반영됩니다. Hold가 만료(2분 경과)되면 자동으로 가용 상태로 복귀합니다.

## 내 예약 필터링

### 필터 옵션

캘린더 화면 상단의 **필터** 아이콘을 탭하면 다음 필터를 적용할 수 있습니다.

| 필터 | 옵션 | 기본값 |
|------|------|--------|
| **표시 범위** | 내 예약만 / 전체 슬롯 | 내 예약만 |
| **예약 상태** | 대기, 승인, 체크인, 완료, 취소, 거절 | 대기 + 승인 |
| **스튜디오** | 전체 / 특정 스튜디오 선택 | 전체 |

### 내 예약만 보기

**내 예약만** 필터를 활성화하면 내가 예약한 슬롯만 캘린더에 표시됩니다. 다른 사용자의 예약은 숨겨지고, 가용 슬롯도 표시되지 않습니다.

### 상태별 필터링

특정 상태만 보고 싶을 때 유용합니다.

- **대기 + 승인**: 아직 이용하지 않은 예정된 예약만 표시
- **완료**: 지난 이용 내역 확인
- **취소 + 거절**: 취소된 예약 이력 확인

### 스튜디오별 필터링

여러 스튜디오를 이용하는 경우, 특정 스튜디오의 예약만 골라볼 수 있습니다. 필터에서 스튜디오를 선택하면 해당 스튜디오의 예약만 캘린더에 표시됩니다.

## 캘린더에서 바로 예약

일간 뷰 또는 주간 뷰에서 **가용(파란색)** 슬롯을 직접 탭하면 예약 화면으로 바로 이동할 수 있습니다. 날짜와 시간이 자동으로 채워져 있어 빠르게 예약을 진행할 수 있습니다.

## 캘린더 동기화

StudioGo 캘린더의 예약 정보를 외부 캘린더 앱과 동기화할 수 있습니다.

### 지원 캘린더

- Google 캘린더
- Apple 캘린더
- Outlook

### 동기화 설정

1. **설정** > **캘린더 동기화**로 이동합니다
2. 연동할 캘린더 서비스를 선택합니다
3. 해당 서비스에 로그인하여 권한을 허용합니다
4. 동기화가 완료되면 승인된 예약이 외부 캘린더에 자동으로 등록됩니다
