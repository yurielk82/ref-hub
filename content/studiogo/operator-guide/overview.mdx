# 운영자 개요

StudioGo 운영자(OPERATOR) 역할과 접근 방법을 안내합니다.

## 운영자 역할

운영자는 창고 기반 라이브커머스 스튜디오를 운영하는 담당자입니다. 셀러(MEMBER)의 예약 요청을 검토하고 승인하며, 현장 체크인 확인과 포장·출고 업무를 처리합니다.

| 기능 | 설명 |
|------|------|
| **예약 관리** | 셀러의 예약 요청 승인·거절, 일괄 승인 |
| **체크인 관리** | QR 스캔·PIN·수동 입력으로 입장 확인 및 체크아웃 처리 |
| **포장/출고 관리** | 방송 후 상품 포장부터 배송 완료까지 단계별 상태 관리 |
| **회원 조회** | 가입한 셀러 회원 검색 및 상태 확인 |

## 권한 범위

| 기능 | MEMBER (셀러) | OPERATOR (운영자) | ADMIN (관리자) |
|------|:-----------:|:----------------:|:-------------:|
| 예약 신청 | O | - | O |
| 예약 승인/거절 | - | O | O |
| 체크인 처리 | - | O | O |
| 포장·출고 상태 변경 | - | O | O |
| 회원 조회 | - | O | O |
| 회원 정지/복구 | - | - | O |
| 스튜디오 설정 | - | - | O |
| 시스템 설정 | - | - | O |

## 접근 방법

### 웹 대시보드

웹 플랫폼(studiogo.vercel.app)은 운영자의 주력 도구입니다. 테이블 형태의 데이터 열람, 일괄 승인, 출고 상태 일괄 변경 등 대량 처리 작업에 적합합니다.

1. [studiogo.vercel.app](https://studiogo.vercel.app)에 접속합니다
2. **카카오 로그인** 버튼을 클릭합니다
3. 로그인 후 자동으로 **운영자 대시보드**로 이동합니다

OPERATOR 권한이 없는 계정으로 로그인하면 권한 없음 페이지가 표시됩니다. 관리자에게 권한 부여를 요청하세요.

### 모바일 앱

모바일 앱은 현장 체크인 처리에 특화되어 있습니다. QR 카메라 스캔 기능을 사용하려면 반드시 앱을 이용해야 합니다.

1. App Store 또는 Google Play에서 **StudioGo**를 설치합니다
2. **카카오 로그인**으로 로그인합니다
3. 하단 탭에서 **운영** 탭으로 이동합니다

## 대시보드 구성

로그인 직후 표시되는 대시보드에서 오늘의 운영 현황을 한눈에 확인할 수 있습니다.

| 항목 | 설명 |
|------|------|
| **오늘 예약** | 오늘 승인된 예약 건수 |
| **승인 대기** | 검토가 필요한 예약 요청 건수 |
| **활성 스튜디오** | 현재 운영 중인 스튜디오 수 |
| **오늘 체크인** | 오늘 처리 완료된 체크인 건수 |

대시보드 하단의 **최근 예약** 테이블에는 최근 10건의 예약이 표시됩니다. **빠른 이동** 버튼으로 예약 관리, 회원 관리, 체크인 화면으로 즉시 이동할 수 있습니다.

## 알림 수신

운영자는 다음 채널로 실시간 알림을 받습니다.

| 알림 유형 | 채널 |
|----------|------|
| 새로운 예약 요청 | 카카오 알림톡, 앱 푸시, 인앱 알림 |
| 셀러 예약 취소 | 카카오 알림톡, 인앱 알림 |
| 노쇼 감지 (체크인 없이 슬롯 종료) | 카카오 알림톡 |
