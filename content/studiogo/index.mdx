# StudioGo

스튜디오 예약 관리 시스템

## 개요

| 항목 | 내용 |
|------|------|
| **스택** | Expo (React Native) + Hono (모노레포) |
| **DB** | 미정 |
| **호스팅** | 미정 |
| **인증** | 미정 |

## 주요 기능

- 스튜디오 예약/취소
- 예약 현황 캘린더
- 알림 기능
