# 시작하기

CoreRx 대시보드에 처음 접속하는 사용자를 위한 가이드입니다. 접속 방법, 로그인, 역할별 권한, 대시보드 진입까지 안내합니다.

## 접속 방법

CoreRx 대시보드는 웹 브라우저에서 접속합니다.

| 항목 | 내용 |
|------|------|
| 접속 URL | 관리자가 안내한 Superset 대시보드 주소 |
| 권장 브라우저 | Chrome 120+, Edge 120+, Safari 17+ |
| 화면 해상도 | 1920x1080 이상 권장 (태블릿 이상) |
| 네트워크 | 사내 VPN 또는 허용된 네트워크에서 접속 |

> **주의**: 모바일 브라우저에서는 차트 레이아웃이 최적화되지 않습니다. 데스크톱 또는 태블릿 환경을 권장합니다.

## 로그인

CoreRx는 Apache Superset의 내장 인증 시스템을 사용합니다.

1. 대시보드 URL에 접속하면 로그인 화면이 표시됩니다.
2. 관리자에게 발급받은 계정 또는 **Google OAuth**로 로그인합니다.
3. 최초 로그인 시 영업관리팀의 승인이 필요할 수 있습니다. 승인 완료 시 이메일로 알림을 받습니다.

### Google OAuth 로그인

1. 로그인 화면에서 **Sign in with Google** 버튼을 클릭합니다.
2. 회사 Google Workspace 계정을 선택합니다.
3. 최초 로그인 시 관리자 승인 대기 상태가 되며, 승인 후 역할에 맞는 대시보드에 접근할 수 있습니다.

## 역할별 접근 권한

로그인 후 사용자의 역할에 따라 볼 수 있는 탭과 데이터 범위가 달라집니다.

| 역할 | 경영 요약 | 제품 성과 | 거래처/지역 | 시장 분석 | CSO 성과 | 데이터 관리 | 시스템 관리 |
|------|-----------|-----------|-------------|-----------|----------|-------------|-------------|
| 경영진 | 전체 조회 | 전체 조회 | 전체 조회 | 전체 조회 | 전체 조회 | - | - |
| 영업관리자 | 전체 조회 | 전체 조회 | 전체 조회 | 전체 조회 | 편집 가능 | 편집 가능 | 편집 가능 |
| 영업사원 | 본인 영업소 | 본인 영업소 | 본인 영업소 | 본인 영업소 | - | - | - |
| CSO | - | - | - | - | 본인 CSO | - | - |

### 본인 데이터 필터링

- **영업사원**: 소속 영업소(부서)에 해당하는 거래처 데이터만 표시됩니다. 다른 영업소의 실적은 볼 수 없습니다.
- **CSO**: 본인이 소속된 CSO 관리업체의 성과 데이터만 표시됩니다. 타 CSO 업체의 데이터는 볼 수 없습니다.

## 대시보드 진입

로그인 후 대시보드에 진입하는 방법입니다.

1. 로그인 성공 후 Superset 메인 화면이 표시됩니다.
2. 상단 메뉴에서 **대시보드** 목록을 클릭합니다.
3. **CoreRx KPI 대시보드**를 선택하면 7개 탭이 있는 대시보드 화면으로 진입합니다.
4. 기본적으로 **경영 요약** 탭이 먼저 표시됩니다.

> **주의**: 대시보드가 목록에 보이지 않으면 관리자에게 접근 권한을 요청하세요. 역할 승인이 완료되지 않았을 수 있습니다.
