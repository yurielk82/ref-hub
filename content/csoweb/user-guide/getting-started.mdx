# 시작하기

CSO 정산 포털에 가입하고 로그인하기까지의 전체 과정을 안내합니다. 회원가입부터 최초 로그인, 비밀번호 재설정까지 순서대로 따라하세요.

## 회원가입

CSO 정산 포털은 관리자 승인제로 운영됩니다. 가입 신청 후 관리자(영업관리팀)의 승인이 완료되어야 로그인할 수 있습니다.

### 가입 절차 흐름

전체 프로세스는 다음과 같습니다.

1. 로그인 화면 하단의 **"회원가입"** 링크를 클릭합니다.
2. 사업자번호를 포함한 업체 정보를 입력합니다.
3. **"회원가입 신청"** 버튼을 클릭합니다.
4. 관리자가 신청 내용을 검토하고 승인합니다.
5. 승인 완료 이메일을 수신한 후 로그인이 가능합니다.

### 입력 항목

회원가입 화면에서 아래 정보를 순서대로 입력합니다.

| 항목 | 필수 | 설명 |
|------|:----:|------|
| 사업자번호 | O | 10자리 숫자 입력 시 국세청에서 자동 인증 |
| 업체명 | O | 사업자등록증 상의 회사명 |
| 대표자명 | O | 사업자등록증 상의 대표자 이름 |
| 주소 | O | **"주소 검색"** 버튼 클릭 → 도로명 주소 선택 → 상세 주소 입력 |
| 연락처1 | O | 대표 전화번호 (입력 시 자동 포맷 적용) |
| 연락처2 | | 추가 연락처 (선택사항) |
| 이메일 | O | 정산서 알림 및 비밀번호 재설정에 사용되는 이메일 |
| 이메일2 | | 추가 이메일 (선택사항) |
| 비밀번호 | O | 영문 + 숫자 조합, 8자 이상 |
| 비밀번호 확인 | O | 위에서 입력한 비밀번호와 동일하게 재입력 |

### 국세청 사업자번호 인증

사업자번호 10자리를 입력하면 국세청 NTS(National Tax Service) API를 통해 실시간으로 사업자 상태가 자동 인증됩니다. 인증 결과에 따라 가입 가능 여부가 결정됩니다.

| 인증 결과 | 화면 표시 | 가입 가능 여부 |
|-----------|----------|:--------------:|
| 계속사업자 | 녹색 체크 아이콘 | O |
| 휴업자 | 주황색 경고 아이콘 | X |
| 폐업자 | 빨간색 경고 아이콘 | X |

**계속사업자**로 확인된 경우에만 가입 신청을 진행할 수 있습니다. 휴업자 또는 폐업자로 표시되는 경우, 국세청에 사업자 상태를 확인한 후 정상화되면 다시 시도하세요.

> **주의**: 사업자번호는 가입 후 변경할 수 없습니다. 정확한 번호를 입력했는지 반드시 확인하세요.

### 관리자 승인 대기

회원가입 신청이 완료되면 관리자(영업관리팀)에게 신청 알림 이메일이 자동 발송됩니다. 이후 아래 절차를 거칩니다.

1. **관리자 검토**: 사업자번호 유효성, CSO 계약 관계, 중복 가입 여부를 확인합니다.
2. **승인 또는 거부**: 관리자가 승인하면 등록된 이메일로 승인 완료 안내 메일이 발송됩니다. 거부 시 거부 사유가 포함된 안내 메일이 발송됩니다.
3. **로그인**: 승인 완료 메일을 수신한 후부터 로그인할 수 있습니다.

> **주의**: 승인이 완료되기 전까지는 사업자번호와 비밀번호를 입력해도 로그인할 수 없습니다. 승인까지 영업일 기준 1~2일이 소요될 수 있습니다.

## 로그인

### 일반 로그인

1. 브라우저에서 CSO 정산 포털 주소에 접속합니다.
2. **사업자번호**를 입력합니다 (하이픈은 자동 입력됩니다).
3. **비밀번호**를 입력합니다.
4. **"로그인"** 버튼을 클릭합니다.

### 최초 로그인 시 필수 단계

관리자가 임시 비밀번호를 발급하여 계정을 생성한 경우, 최초 로그인 시 반드시 아래 2단계를 거쳐야 합니다. 두 단계를 모두 완료해야 정산서 조회 화면으로 진입할 수 있습니다.

**1단계 — 비밀번호 변경**

임시 비밀번호는 보안상 즉시 변경해야 합니다. 새 비밀번호는 영문 + 숫자 조합, 8자 이상이어야 합니다.

1. 자동으로 비밀번호 변경 화면이 표시됩니다.
2. **새 비밀번호**를 입력합니다.
3. **새 비밀번호 확인**에 동일한 비밀번호를 다시 입력합니다.
4. **"비밀번호 변경"** 버튼을 클릭합니다.

**2단계 — 회원 정보 완성**

비밀번호 변경 후 업체 정보를 입력하는 화면이 표시됩니다.

1. 업체명, 대표자명, 주소, 연락처, 이메일을 입력합니다.
2. 모든 필수 항목을 빠짐없이 채웁니다.
3. **"저장"** 버튼을 클릭합니다.

> **주의**: 두 단계를 모두 완료하지 않으면 정산 조회 화면에 접근할 수 없습니다. 브라우저를 닫고 다시 로그인해도 미완료 단계부터 다시 시작됩니다.

## 비밀번호 재설정

비밀번호를 잊은 경우 이메일을 통해 새 비밀번호를 설정할 수 있습니다.

### 재설정 절차

1. 로그인 화면에서 **"비밀번호를 잊으셨나요?"** 링크를 클릭합니다.
2. 가입 시 등록한 **사업자번호**와 **이메일**을 입력합니다.
3. **"재설정 링크 발송"** 버튼을 클릭합니다.
4. 입력한 이메일로 비밀번호 재설정 링크가 발송됩니다.
5. 이메일에서 **"비밀번호 재설정하기"** 버튼을 클릭합니다.
6. 새 비밀번호를 입력하고 **"비밀번호 변경"** 버튼을 클릭합니다.
7. 변경이 완료되면 로그인 화면으로 자동 이동합니다.

### 재설정 링크 유효 조건

| 조건 | 내용 |
|------|------|
| 유효 시간 | 발송 후 **30분** 이내 |
| 사용 횟수 | **1회만** 사용 가능 |
| 보안 | 타인에게 링크를 공유하지 마세요 |

> **주의**: 재설정 링크가 만료되었거나 이미 사용된 경우, 위 절차를 처음부터 다시 진행하세요.

### 이메일이 도착하지 않을 때

- 스팸/정크 메일 폴더를 확인하세요.
- 가입 시 등록한 이메일 주소가 정확한지 확인하세요.
- 그래도 도착하지 않으면 관리자(영업관리팀)에게 비밀번호 초기화를 요청하세요. 관리자가 임시 비밀번호를 발급하면 해당 비밀번호로 로그인 후 새 비밀번호를 설정합니다.

## 화면 구성

로그인 후 상단 메뉴에서 각 기능에 접근할 수 있습니다.

| 메뉴 | 설명 |
|------|------|
| **홈** | 최근 정산 현황 요약 및 공지사항 |
| **정산 조회** | 정산서 상세 데이터 검색, 필터링, 엑셀 다운로드 |
| **월별 합계** | 정산월별 수수료 합산 통계 |
| **내 정보** | 업체 정보 수정 및 비밀번호 변경 |
