# 시작하기

EDI 검증 시스템 접속부터 기본 화면 구성까지 안내합니다.

## 접속 경로

CSO Web 관리자 메뉴에서 **EDI 검증**을 클릭합니다.

```
CSO Web → 관리자 메뉴 → EDI 검증
```

## 권한

EDI 검증 기능은 **관리자 권한**이 필요합니다. 일반 CSO 회원에게는 메뉴가 노출되지 않습니다.

## 화면 구성

EDI 검증 메인 화면은 3단계 스텝 방식으로 구성됩니다:

| 단계 | 이름 | 설명 |
|------|------|------|
| Step 1 | 파일 업로드 | ZIP 파일 선택 및 처방월 지정 |
| Step 2 | CSO 매칭 | 폴더별 CSO 업체 자동/수동 매칭 |
| Step 3 | 처리 진행 | OCR 처리 진행률 및 결과 확인 |

하단에는 **처리 이력** 섹션이 있어 과거 배치 목록과 엑셀 다운로드를 제공합니다.

## 기본 워크플로우

1. ZIP 파일을 드래그앤드롭 또는 클릭으로 업로드
2. 처방월(조회기간) 선택
3. CSO 매칭 확인 후 처리 시작
4. 처리 완료 후 플래그된 건을 검수
5. 배치 확정 및 엑셀 다운로드
