# 시작하기

Union KPI 대시보드에 처음 접속하는 방법을 안내합니다.

## 접속

1. 브라우저에서 [kpi.dvsharp.com](https://kpi.dvsharp.com)에 접속합니다
2. **로그인** 페이지에서 이메일과 비밀번호를 입력합니다
3. 최초 가입 시 **회원가입** 후 관리자 승인을 기다립니다

> **주의**: 관리자가 승인하기 전까지는 대기 화면이 표시됩니다.

## 화면 구성

로그인 후 메인 대시보드가 표시됩니다:

| 영역 | 설명 |
|------|------|
| **사이드바** | 메뉴 네비게이션 — 매출, 수금, 마진 등 각 대시보드로 이동 |
| **헤더** | 검색(⌘K), 사용자 메뉴, 테마 전환 |
| **메인 영역** | 선택한 대시보드의 차트와 데이터 |

### 사이드바 메뉴 구조

- **매출** — 도매 매출, CSO 매출, 흡수율
- **수금** — 수금 현황
- **마진** — 마진 분석
- **영업활동** — 영업 활동 지표
- **공급** — 공급 현황
- **지도** — 지역별 매출 분포
- **준수율** — 컴플라이언스
- **업로드** — 데이터 업로드
- **설정** — AI, 테마, 연결 상태
