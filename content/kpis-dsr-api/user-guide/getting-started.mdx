# 시작하기

KPIS DSR API 시스템 사용을 위한 초기 설정 가이드입니다.

## 로그인

1. 웹 브라우저에서 시스템 주소에 접속합니다
2. **Google 계정으로 로그인** 버튼을 클릭합니다
3. 승인된 Google 계정으로 인증을 완료합니다

## 권한

관리자가 승인한 Google 계정만 접근 가능합니다. 접근 권한이 없는 경우 관리자에게 문의하세요.

## 기본 워크플로우

```
엑셀(갑지+을지) → [업로드] → 자동 코드매핑/규칙 적용
     ↓
[유효성 검증] → 오류 수정 → 최종 확인
     ↓
├→ [수동보고] 엑셀 내보내기 → KPIS 포털 직접 업로드
└→ [자동보고] MA111 API 제출 → MA112 결과 확인
```

## 핵심 용어

| 용어 | 설명 |
|------|------|
| **KPIS** | 의약품관리종합정보센터 |
| **DSR** | Drug Supply Report (의약품 공급내역 보고서) |
| **갑지** | 공급자 보고 문서 (21열) |
| **을지** | 일련번호 문서 (3열) |
| **표준코드** | KPIS 13자리 의약품 식별코드 |
| **공급구분** | 1출고 / 2반품 / 3폐기 / 4정정 / 5취소 |
