# 시장 분석

시장 분석 탭은 UBIST 처방조제 데이터를 기반으로 자사의 시장 내 포지션을 분석합니다. 시장점유율(Market Share, MS%) 추이, 약효군별 경쟁력, 처방-매출 갭을 확인할 수 있습니다.

## 데이터 소스

시장 분석의 모든 데이터는 UBIST(IMS) 처방조제 데이터에서 산출됩니다.

| 항목 | 내용 |
|------|------|
| 데이터 제공 | UBIST (IMS Health) |
| 범위 | 전체 시장 268개 제조사, 13,000여 제품 |
| 갱신 주기 | 분기 1회 (2~3개월 지연) |
| 측정 기준 | 처방조제액 (prescription_amount) |

> **주의**: UBIST 데이터는 2~3개월 지연되므로, 최근 2~3개월의 시장 데이터는 표시되지 않을 수 있습니다. 이는 데이터 오류가 아닌 UBIST의 수집/가공 주기에 따른 정상적인 지연입니다.

## 위젯 구성

| 위젯 | 유형 | 표시 내용 |
|------|------|-----------|
| 시장점유율 추이 | 라인 차트 | 월별 MS% + ERP 매출 오버레이 |
| 약효군별 시장점유율 | 가로 막대 차트 | 약효군(한글)별 자사 MS% |
| 시장 성장률 테이블 | 테이블 | 약효군별 시장/자사 성장률, 점유율 변동 |

## 시장점유율(MS%) 추이

자사의 전체 시장점유율을 월별 라인 차트로 표시합니다. ERP 실매출 라인이 함께 오버레이되어 처방 추세와 실판매의 관계를 확인할 수 있습니다.

### MS% 계산 방식

시장점유율은 다음과 같이 계산됩니다:

- **자사 처방금액**: UBIST 데이터에서 자사(유니온) 제조사의 처방조제액 합계
- **전체 시장 금액**: UBIST 데이터에서 동일 약효군 전체 제조사의 처방조제액 합계
- **MS%** = 자사 처방금액 / 전체 시장 금액 x 100

### 추이 해석

| 패턴 | 의미 | 확인 사항 |
|------|------|-----------|
| MS% 상승 | 시장 내 입지 강화 | 어떤 제품/약효군이 기여하는지 확인 |
| MS% 하락 | 경쟁사 대비 열세 | 경쟁제 출시, 급여 변경, 영업 이슈 점검 |
| MS% 유지 + 시장 성장 | 시장과 동일 속도 성장 | 시장 확대 기회를 추가 포착할 여지 |
| MS% 유지 + 시장 축소 | 시장 대비 선방 | 방어 전략 유효, 신시장 진출 검토 |

## 약효군별 시장점유율

약효군(ATC 분류)별로 자사의 시장점유율을 가로 막대 차트로 표시합니다. 약효군명은 한글로 매핑됩니다.

### 활용법

- **점유율 높은 약효군**: 자사의 핵심 경쟁력 영역. 방어 전략과 점유율 유지가 중요합니다.
- **점유율 낮은 약효군**: 성장 기회 또는 진입 어려움. 시장 규모와 경쟁 강도를 함께 확인합니다.
- 크로스필터로 특정 약효군을 클릭하면 시장 성장률 테이블이 해당 약효군만 표시합니다.

## 경쟁사 비교

UBIST 데이터에는 268개 제조사의 처방 데이터가 포함되어 있어, 약효군별 주요 경쟁사의 점유율을 비교할 수 있습니다.

### 비교 관점

- **직접 경쟁사**: 동일 약효군에서 높은 점유율을 가진 제조사
- **점유율 변동**: 경쟁사의 MS% 변동이 자사와 반대 방향이면 직접적인 고객 이동 가능성
- **시장 진입자**: 최근 점유율이 급증한 신규 진입자는 경쟁 심화 신호

## 처방-매출 갭 분석

UBIST 처방량과 ERP/갑지 실판매량 간의 차이를 분석합니다.

| 갭 상태 | 의미 | 가능한 원인 |
|---------|------|-------------|
| 처방 > 판매 | 처방은 발생했으나 공급이 부족 | 재고 부족, 유통 지연, 대체 제품 조제 |
| 처방 < 판매 | 공급이 처방보다 많음 | 재고 비축, 도매 물량, 반품 미반영 |
| 처방 ≈ 판매 | 정상적 유통 흐름 | 공급과 소비가 균형 |

### 갭 분석 활용

1. 처방 > 판매인 제품은 **공급 확대 기회**입니다. 해당 거래처에 영업 활동을 강화합니다.
2. 처방 < 판매인 제품은 **재고 과잉 위험**입니다. 유통 재고 소진 상황을 모니터링합니다.
3. 갭이 3개월 이상 지속되면 구조적 원인을 파악합니다.

> **주의**: UBIST 데이터와 ERP 데이터의 시점이 2~3개월 차이나므로, 갭 분석 시 동일 시점의 데이터끼리 비교해야 합니다. 필터에서 UBIST 데이터가 존재하는 기간을 선택하세요.

## 시장 성장률 테이블

약효군별 시장 전체와 자사의 성장률을 비교하는 테이블입니다.

| 컬럼 | 설명 |
|------|------|
| 약효군 | ATC 분류 기준 한글 약효군명 |
| 시장 성장률 (YoY%) | 전년 동월 대비 전체 시장 성장률 |
| 자사 성장률 (YoY%) | 전년 동월 대비 자사 처방금액 성장률 |
| 점유율 변동 (%p) | 당월 MS% - 전년 동월 MS% |

### 해석 기준

- **자사 성장률 > 시장 성장률**: 시장 대비 고성장, 점유율 상승
- **자사 성장률 < 시장 성장률**: 시장 대비 저성장, 점유율 하락
- **점유율 변동 양수**: 시장 내 입지 강화
- **점유율 변동 음수**: 시장 내 입지 약화
