# 배포

EV Motor Reliability 배포 가이드입니다.

## 아키텍처

- **API 서버**: FastAPI + Uvicorn (Python)
- **웹 프론트엔드**: Next.js 15 (`web/` 디렉토리)

## API 서버 실행

```bash
cd ev-motor-reliability
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --host 0.0.0.0 --port 8000
```

## 웹 프론트엔드 실행

```bash
cd ev-motor-reliability/web
npm install
npm run build
npm start
```
