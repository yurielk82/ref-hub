# 스튜디오 관리

창고 운영자(OPERATOR)가 StudioGo에서 스튜디오를 등록하고 관리하는 방법을 안내합니다.

## 역할 권한

스튜디오 관리 기능은 **OPERATOR** 이상의 권한이 필요합니다.

| 기능 | MEMBER (BJ) | OPERATOR (운영자) | ADMIN (관리자) |
|------|:-----------:|:----------------:|:-------------:|
| 스튜디오 조회 | O | O | O |
| 스튜디오 등록 | - | O | O |
| 스튜디오 수정 | - | 본인 소유만 | O |
| 스튜디오 삭제 | - | 본인 소유만 | O |
| 예약 승인/거절 | - | 본인 스튜디오만 | O |
| 체크인 처리 | - | 본인 스튜디오만 | O |
| 알림톡 설정 | - | 본인 스튜디오만 | O |

## 스튜디오 등록

### 등록 화면 진입

하단 탭의 **관리** > **스튜디오 등록** 버튼을 탭합니다.

### 기본 정보 입력

| 필드 | 필수 | 설명 | 예시 |
|------|:----:|------|------|
| **스튜디오명** | O | 스튜디오 이름 | 동대문 A스튜디오 |
| **주소** | O | 스튜디오 소재지 (주소 검색) | 서울시 중구 을지로 123 |
| **상세 주소** | O | 건물 내 위치 | 3층 301호 |
| **전화번호** | O | 운영자 연락처 | 02-1234-5678 |
| **소개** | O | 스튜디오 설명 (최대 500자) | 의류 라이브 전문 스튜디오 |
| **수용 인원** | O | 최대 이용 가능 인원 | 5명 |
| **사진** | O | 스튜디오 사진 (최소 3장, 최대 10장) | 내부 전경, 설비 등 |

### 설비 정보

스튜디오에 구비된 설비를 체크리스트에서 선택합니다. 목록에 없는 설비는 직접 입력할 수 있습니다.

- 조명 (링라이트, 소프트박스 등)
- 행거 (이동식, 고정식)
- 카메라 (FHD, 4K)
- 삼각대/짐벌
- Wi-Fi (유선 백업 포함 여부)
- 모니터
- 스팀다리미
- 탈의실
- 주차 가능 여부

### 이용 요금 설정

| 항목 | 설명 |
|------|------|
| **시간당 요금** | 기본 슬롯 1개당 이용 요금 |
| **일일 요금** | 하루 전체 이용 시 할인 요금 (선택) |
| **추가 인원 요금** | 기본 인원 초과 시 1인당 추가 요금 (선택) |

### 등록 완료

모든 정보를 입력한 후 **등록하기** 버튼을 탭합니다. ADMIN의 검토를 거쳐 승인되면 스튜디오가 검색 결과에 노출됩니다.

## 스튜디오 수정

### 수정 화면 진입

**관리** > **내 스튜디오** > 수정할 스튜디오 선택 > **편집** 아이콘을 탭합니다.

### 수정 가능 항목

등록 시 입력한 모든 항목을 수정할 수 있습니다. 단, 이미 예약이 잡혀 있는 시간대에 영향을 주는 변경은 주의가 필요합니다.

| 변경 유형 | 기존 예약 영향 | 주의 사항 |
|----------|:------------:|----------|
| 사진/소개 수정 | 없음 | 자유롭게 수정 가능 |
| 요금 변경 | 없음 | 기존 예약은 확정 당시 요금 유지 |
| 설비 변경 | 없음 | 목록만 갱신 |
| 운영 시간 축소 | **있음** | 기존 예약과 겹치면 수정 불가 |
| 슬롯 간격 변경 | **있음** | 해당일 이후 예약이 없을 때만 가능 |

## 스튜디오 삭제

**관리** > **내 스튜디오** > 대상 스튜디오 선택 > **설정** > **스튜디오 삭제**를 탭합니다.

### 삭제 조건

- 미래 날짜에 **승인 상태 예약**이 없어야 삭제할 수 있습니다
- 대기 상태 예약이 있으면 자동 취소 후 삭제됩니다
- 삭제된 스튜디오의 과거 예약 기록은 **90일간** 보관됩니다

### 일시 중지

삭제 대신 **일시 중지**를 선택하면 신규 예약만 차단하고 기존 예약은 유지됩니다. **관리** > **내 스튜디오** > **설정** > **일시 중지**에서 활성화합니다.

## 운영 시간 설정

### 요일별 운영 시간

**관리** > **내 스튜디오** > **운영 시간**에서 요일별로 운영 시간을 설정합니다.

| 설정 | 설명 | 예시 |
|------|------|------|
| **운영 요일** | 운영하는 요일 선택 | 월~토 |
| **시작 시간** | 첫 슬롯 시작 시간 | 09:00 |
| **종료 시간** | 마지막 슬롯 종료 시간 | 22:00 |
| **휴게 시간** | 예약 불가 시간대 (선택) | 12:00~13:00 |

### 특정일 차단

공휴일, 정비일 등 특정 날짜에 예약을 받지 않으려면 **운영 시간** > **특정일 관리**에서 날짜를 지정합니다.

1. **특정일 추가** 버튼을 탭합니다
2. 날짜를 선택합니다 (범위 선택 가능)
3. 사유를 입력합니다 (선택)
4. **저장** 버튼을 탭합니다

차단된 날짜는 캘린더에 빗금 표시되며 예약이 불가합니다.

## 슬롯 간격 설정

**관리** > **내 스튜디오** > **슬롯 설정**에서 슬롯의 시간 간격을 설정합니다.

### 간격 옵션

| 간격 | 슬롯 예시 (09:00~18:00 기준) | 용도 |
|------|---------------------------|------|
| **30분** | 09:00, 09:30, 10:00, ... | 짧은 촬영, 유연한 일정 |
| **1시간** | 09:00, 10:00, 11:00, ... | 일반적인 라이브 방송 |
| **2시간** | 09:00, 11:00, 13:00, ... | 장시간 촬영 |
| **사용자 정의** | 자유 설정 | 특수 운영 |

### 간격 변경 시 주의

- 슬롯 간격을 변경하면 **변경일 이후**의 슬롯이 재생성됩니다
- 이미 예약이 있는 날짜의 간격은 변경할 수 없습니다
- 간격 변경 시 기존 Hold는 자동 해제됩니다

## 예약 승인/거절

### 예약 요청 확인

새로운 예약 요청이 들어오면 다음 채널로 알림을 받습니다.

- **앱 푸시**: "새로운 예약 요청이 있습니다"
- **카카오 알림톡**: 예약자 정보, 날짜, 시간대 포함
- **인앱 알림**: 알림 탭에 기록

### 승인 절차

1. **관리** > **예약 관리**에서 **대기** 탭을 탭합니다
2. 예약 요청을 확인합니다 (예약자, 날짜, 시간, 이용 목적)
3. **승인** 또는 **거절** 버튼을 탭합니다
4. 거절 시 사유를 입력합니다 (필수)

### 승인/거절 기준

| 고려 사항 | 설명 |
|----------|------|
| 슬롯 충돌 | 같은 시간대에 이미 승인된 예약이 있는지 확인 |
| 이용자 이력 | 과거 노쇼, 시설 훼손 이력 확인 |
| 이용 목적 | 스튜디오 용도에 적합한지 확인 |

### 자동 승인

예약 요청을 자동으로 승인하도록 설정할 수 있습니다. **관리** > **내 스튜디오** > **설정** > **자동 승인**을 활성화하면 접수 즉시 승인 처리됩니다.

## 체크인 처리

### 체크인 방법

예약 당일, BJ가 스튜디오에 도착하면 운영자가 체크인을 처리합니다.

1. **관리** > **예약 관리**에서 **오늘 예약** 탭을 탭합니다
2. 도착한 BJ의 예약을 확인합니다
3. **체크인** 버튼을 탭합니다
4. 체크인이 완료되면 예약 상태가 **체크인**으로 변경됩니다

### QR 체크인

BJ의 앱에 표시되는 QR 코드를 운영자 앱의 **QR 스캔** 기능으로 읽어 체크인할 수 있습니다.

### 이용 종료

슬롯 종료 시간이 되면 예약 상태가 자동으로 **완료**로 변경됩니다. 조기 퇴실 시 운영자가 수동으로 **이용 종료** 버튼을 탭할 수 있습니다.

## 알림톡 설정

### 알림톡 종류

운영자가 수신하는 카카오 알림톡 종류와 설정 방법입니다.

| 알림 유형 | 기본값 | 변경 가능 | 내용 |
|----------|:------:|:--------:|------|
| 새 예약 요청 | ON | O | 신규 예약 접수 알림 |
| 예약 취소 | ON | O | BJ가 예약을 취소한 경우 |
| 체크인 알림 | ON | O | BJ가 체크인한 경우 |
| 일일 요약 | OFF | O | 다음 날 예약 목록 요약 (전날 저녁 발송) |
| 노쇼 알림 | ON | - | 체크인 없이 슬롯 종료 시 알림 (비활성화 불가) |

### 알림 수신 설정

**관리** > **설정** > **알림 설정**에서 각 알림 유형의 수신 여부를 변경할 수 있습니다.

### 알림톡 수신 번호

카카오 알림톡은 카카오 로그인에 사용한 전화번호로 발송됩니다. 수신 번호를 변경하려면 **설정** > **내 정보** > **연락처**에서 전화번호를 수정합니다.

## 대시보드

**관리** 화면에 진입하면 운영 대시보드가 표시됩니다.

### 주요 지표

| 지표 | 설명 |
|------|------|
| **오늘 예약** | 오늘 승인된 예약 건수 |
| **대기 중** | 승인 대기 중인 예약 건수 |
| **이번 주 이용률** | (승인 슬롯 / 전체 가용 슬롯) x 100 |
| **이번 달 매출** | 확정된 예약의 총 요금 합계 |
| **노쇼율** | (노쇼 건수 / 전체 승인 건수) x 100 |
