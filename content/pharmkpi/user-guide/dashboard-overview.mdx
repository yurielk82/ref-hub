# 대시보드 개요

메인 대시보드에서 제공하는 종합 KPI 현황을 설명합니다.

## 메인 대시보드

로그인 후 첫 화면에서 전체 매출 현황을 한눈에 확인할 수 있습니다.

### 주요 지표

| 지표 | 설명 |
|------|------|
| **도매 매출** | 도매사를 통한 제품 출고 금액 |
| **CSO 매출** | CSO 업체를 통한 영업 매출 |
| **수금율** | 매출 대비 실제 수금 비율 |
| **흡수율** | 투입 금액 대비 매출 회수 비율 |

## 기간 선택

각 대시보드에서 조회 기간을 변경할 수 있습니다:

- **월별 선택** — 특정 월의 데이터 조회
- **기간 비교** — 전월 대비, 전년 동월 대비

> **주의**: 과거 월 데이터는 캐시에서 제공되므로 빠르게 조회됩니다. 당월 데이터만 실시간으로 동기화됩니다.

## 차트 활용

- 차트 위에 마우스를 올리면 상세 수치가 표시됩니다
- 범례를 클릭하면 해당 항목을 표시/숨김 전환할 수 있습니다
