# 대시보드

대시보드는 월간 보고 현황을 한눈에 파악하고, 주요 작업으로 빠르게 이동할 수 있는 시작 화면입니다. 사이드바의 **대시보드** 메뉴 또는 앱 로고를 클릭하면 표시됩니다.

## 월별 탭 구조

대시보드 상단에 월별 탭이 표시됩니다. 좌측/우측 화살표 버튼으로 과거 월의 통계를 조회할 수 있습니다. 기본값은 현재 월이며, 탭을 전환하면 아래의 모든 카드와 차트가 해당 월의 데이터로 갱신됩니다.

## 공동인증서 만료 경고

대시보드 최상단에 공동인증서 만료 경고 배너가 표시됩니다. 인증서 만료일 기준으로 단계별 경고가 제공됩니다.

| 잔여 기간 | 경고 수준 | 배너 색상 |
|-----------|----------|----------|
| 30일 이내 | 사전 경고 | 노란색 |
| 7일 이내 | 긴급 경고 | 주황색 |
| 1일 이내 또는 만료 | 즉시 갱신 필요 | 빨간색 |

> **주의**: 공동인증서가 만료되면 KPIS API 자동 제출(MA111~MA114)이 불가능합니다. 만료 전에 반드시 새 인증서를 등록해야 합니다.

## KPI 카드

대시보드 중앙에 해당 월의 핵심 지표가 카드 형태로 표시됩니다.

| 카드 | 표시 내용 | 설명 |
|------|----------|------|
| **총 행 수** | 갑지 전체 건수 | 해당 월에 업로드된 모든 공급내역의 총 행 수 |
| **보고 완료** | 등록 완료 건수 | KPIS에 `registered` 상태로 확인된 건수 |
| **오류** | 검증 오류 건수 | 제출 불가능한 오류가 있는 행 수 |
| **미보고** | 미제출 건수 | 아직 제출하지 않은 행 수 |
| **보고율** | 백분율 (%) | 보고 완료 건수 / 총 행 수 × 100 |

각 카드를 클릭하면 해당 데이터의 상세 화면으로 이동합니다.

## 차트

### 일별 보고 차트

막대 그래프로 해당 월의 일별 보고 건수를 시각화합니다. 각 막대는 하루 단위의 보고 건수를 나타내며, 보고 완료/미완료를 색상으로 구분합니다.

### 공급구분별 차트

파이 차트로 공급구분(출고/반품/폐기/정정/취소)별 비율을 시각화합니다. 각 공급구분의 건수와 백분율을 확인할 수 있습니다.

## 최근 보고 날짜

최근에 데이터를 업로드하거나 보고한 날짜 목록이 표시됩니다. 날짜를 클릭하면 해당 날짜의 데이터 편집 화면으로 바로 이동합니다.

## 빠른 작업

대시보드 하단에 자주 사용하는 기능으로 바로 이동하는 버튼이 제공됩니다.

| 버튼 | 이동 위치 | 설명 |
|------|----------|------|
| **파일 업로드** | 업로드 화면 | 새 갑지/을지 엑셀 파일을 업로드합니다 |
| **데이터 관리** | 관리 화면 | 날짜별 데이터 조회 및 삭제를 수행합니다 |

## 화면 이동

대시보드에서 각 화면으로의 이동 경로는 다음과 같습니다.

| 목적 | 이동 방법 |
|------|----------|
| 특정 날짜의 데이터 편집 | 최근 보고 날짜 목록에서 날짜 클릭 |
| 새 파일 업로드 | **파일 업로드** 빠른 작업 버튼 클릭 |
| 날짜별 데이터 관리 | **데이터 관리** 빠른 작업 버튼 클릭 |
| 특정 공급구분 데이터 | 사이드바의 공급구분 탭 (출고/반품/폐기/정정/취소) 클릭 |
