# 거래처/지역 분석

거래처/지역 분석 탭은 영업소(지역)별 매출 분포와 개별 거래처의 실적을 분석합니다. 매출 집중도를 파악하고, 거래처 유형(병원/약국)별 비중을 확인할 수 있습니다.

## 위젯 구성

| 위젯 | 유형 | 표시 내용 |
|------|------|-----------|
| 영업소별 매출 비중 | 파이 차트 | 영업소(지역) 단위 매출 비율 |
| TOP 20 거래처 | 가로 막대 차트 | 매출 상위 20개 거래처와 금액 (백만원) |
| 거래처 상세 현황 | 테이블 | 연월, 거래처명, 매출, MoM%, YoY% |

## 영업소별 매출 비중

파이 차트로 영업소(부서) 단위의 매출 비율을 표시합니다. 각 조각은 하나의 영업소를 나타내며, 비율이 클수록 해당 영업소의 매출 비중이 높습니다.

### 확인 포인트

- **매출 집중도**: 특정 영업소에 매출이 과도하게 집중되면 리스크 요인입니다. 해당 영업소의 핵심 거래처 이탈 시 전사 매출에 큰 영향을 줍니다.
- **소규모 영업소**: 매출 비중이 극히 낮은 영업소는 영업 활동 강화 또는 구조 조정 검토 대상입니다.

### 크로스필터 활용

파이 조각을 클릭하면 해당 영업소의 거래처만 하단 차트와 테이블에 필터링됩니다.

## TOP 20 거래처

매출 금액 기준 상위 20개 거래처를 가로 막대 차트로 표시합니다.

### 거래처 유형

제약사의 거래처는 크게 다음 유형으로 구분됩니다:

| 유형 | 설명 | 결제 특성 |
|------|------|-----------|
| 종합병원 | 상급종합, 종합병원 | 60~90일 결제, 비교적 안정 |
| 병원 | 병원, 의원 | 30~60일 결제 |
| 약국 | 조제약국, 일반약국 | 30~45일 결제 |
| 도매상 | 백제, 지오영 등 | 30~60일 결제, 물량 큼 |

### 활용법

- **도매상 비중**: 도매상 매출 비중이 과도하면 최종 소비처 파악이 어려워집니다.
- **대형 거래처 의존도**: TOP 5 거래처가 전체 매출의 50% 이상이면 거래처 다변화가 필요합니다.
- **신규 거래처**: 최근 새로 진입한 거래처는 영업 확대 기회입니다.

## 거래처 상세 현황 테이블

전체 거래처의 월별 매출과 성장률을 상세하게 표시합니다.

| 컬럼 | 설명 |
|------|------|
| 연월 | 매출 발생 연월 (YYYY-MM) |
| 거래처명 | 거래처 한글명 |
| 매출 | 공급금액 합계 (백만원) |
| MoM% | 전월 대비 성장률 |
| YoY% | 전년 동월 대비 성장률 |

### 활용법

- 테이블 헤더를 클릭하여 정렬 기준을 변경할 수 있습니다.
- **MoM%** 컬럼으로 정렬하면 급성장/급락 거래처를 빠르게 파악합니다.
- **YoY%** 컬럼으로 정렬하면 연간 기준 성장/위축 거래처를 확인합니다.
- 기간 필터와 조합하면 특정 분기의 거래처별 실적을 추출할 수 있습니다.

> **주의**: 영업사원 역할로 로그인한 경우, 본인 소속 영업소의 거래처만 테이블에 표시됩니다.
