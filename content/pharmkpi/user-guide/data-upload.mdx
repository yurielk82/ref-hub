# 데이터 업로드

엑셀 데이터를 업로드하여 대시보드에 반영하는 방법을 안내합니다.

## 업로드 페이지 접속

사이드바에서 **업로드**를 선택합니다.

## 일반 업로드

1. **업로드** 페이지에서 **파일 선택** 버튼을 클릭합니다
2. 엑셀 파일(.xlsx)을 선택합니다
3. 파일 검증 결과를 확인합니다
4. **업로드** 버튼을 클릭하여 DB에 저장합니다

> **주의**: 업로드 양식이 맞지 않으면 검증 단계에서 오류가 표시됩니다. **템플릿 다운로드** 메뉴에서 올바른 양식을 받아 사용하세요.

## 흡수율 업로드

사이드바에서 **업로드 > 흡수율**을 선택합니다.

흡수율은 별도 양식을 사용합니다:

1. **템플릿 다운로드** 메뉴에서 흡수율 전용 양식을 다운로드합니다
2. 데이터를 입력합니다
3. **흡수율 업로드** 페이지에서 파일을 업로드합니다

## 업로드 이력

사이드바에서 **업로드 > 이력**을 선택하면 과거 업로드 기록을 확인할 수 있습니다:

| 항목 | 설명 |
|------|------|
| **업로드 일시** | 파일이 업로드된 시각 |
| **파일명** | 업로드한 파일 이름 |
| **상태** | 성공/실패 여부 |
| **처리 건수** | 처리된 레코드 수 |

## 템플릿 다운로드

사이드바에서 **업로드 > 템플릿**을 선택하면 각 데이터 유형별 엑셀 양식을 다운로드할 수 있습니다.
