# 자주 묻는 질문

CSO 정산 포털 사용 중 자주 발생하는 문제와 해결 방법을 안내합니다.

## 로그인

### 로그인이 안 됩니다

**원인 1: 사업자번호 오입력**

사업자번호는 하이픈(-) 없이 10자리 숫자로 입력해야 합니다.

| 입력 방식 | 예시 | 결과 |
|----------|------|------|
| 올바른 입력 | `1234567890` | 로그인 성공 |
| 하이픈 포함 | `123-45-67890` | 로그인 실패 |
| 공백 포함 | `123 456 7890` | 로그인 실패 |

**원인 2: 계정 미승인**

회원가입 신청 후 관리자 승인이 완료되지 않은 상태에서는 로그인할 수 없습니다. 가입 신청 후 영업일 기준 1~2일 후에 다시 시도하세요. 급한 경우 관리자(영업관리팀)에게 승인 상태를 문의하세요.

**원인 3: 비밀번호 오류**

비밀번호는 대소문자를 구분합니다. Caps Lock이 켜져 있지 않은지 확인하세요. 비밀번호를 분실한 경우 아래 [비밀번호를 잊었습니다](#비밀번호를-잊었습니다) 항목을 참조하세요.

---

## 정산 데이터

### 정산 데이터가 보이지 않습니다

정산 조회 화면에 데이터가 표시되지 않는 주요 원인은 다음과 같습니다.

**원인 1: CSO 매칭 미완료**

가장 흔한 원인입니다. CSO 매칭이 완료되어야 자사의 정산 데이터가 표시됩니다. 관리자에게 매칭 상태를 확인 요청하세요.

- 매칭 상태 확인: 관리자에게 사업자번호와 회사명을 전달하여 매칭 여부를 확인합니다.
- 상세 안내: [CSO 매칭](/csoweb/user-guide/equipment) 페이지 참조

**원인 2: 해당 처방월 데이터 없음**

선택한 처방월에 자사의 정산 데이터가 존재하지 않을 수 있습니다. 다른 처방월을 선택하여 확인하세요.

**원인 3: 데이터 미업로드**

관리자가 해당 처방월의 정산 엑셀을 아직 업로드하지 않았을 수 있습니다. 관리자에게 업로드 일정을 문의하세요.

## 엑셀 다운로드

### 엑셀 다운로드가 안 됩니다

**원인 1: 브라우저 팝업 차단**

일부 브라우저에서 파일 다운로드를 팝업으로 인식하여 차단할 수 있습니다.

| 브라우저 | 해제 방법 |
|---------|----------|
| Chrome | 주소창 우측 차단 아이콘 클릭 → "항상 허용" 선택 |
| Edge | 주소창 우측 차단 아이콘 클릭 → "허용" 선택 |
| Safari | 환경설정 → 웹사이트 → 팝업 창 → 해당 사이트 "허용" |

**원인 2: 데이터 없음**

필터 조건에 맞는 데이터가 없으면 다운로드할 파일이 생성되지 않습니다. 필터 조건을 확인하거나 필터를 해제한 후 다시 시도하세요.

**원인 3: 대용량 데이터**

데이터가 매우 많은 경우 파일 생성에 시간이 걸릴 수 있습니다. 버튼 클릭 후 잠시 기다려 주세요. 반복적으로 실패하면 필터를 적용하여 데이터 범위를 줄인 후 시도하세요.

## 비밀번호

### 비밀번호를 잊었습니다

비밀번호 재설정 절차는 다음과 같습니다.

1. 로그인 화면에서 **비밀번호 찾기** 링크를 클릭합니다.
2. 가입 시 등록한 **사업자번호**와 **이메일**을 입력합니다.
3. 해당 이메일로 비밀번호 재설정 링크가 발송됩니다.
4. 이메일의 링크를 클릭하여 새 비밀번호를 설정합니다.

> 재설정 이메일이 도착하지 않으면 스팸 폴더를 확인하세요. 그래도 없다면 등록된 이메일 주소가 올바른지 관리자에게 확인을 요청하세요.

## 회원가입

### 회원가입 후 언제 사용 가능한가요

회원가입 신청 후 아래 두 단계가 완료되어야 정산 데이터를 조회할 수 있습니다.

| 단계 | 소요 시간 | 완료 후 가능한 것 |
|------|----------|----------------|
| 1. 관리자 승인 | 영업일 1~2일 | 로그인 가능 |
| 2. CSO 매칭 | 승인 후 영업일 1~2일 | 정산 데이터 조회 가능 |

- 승인이 완료되면 등록한 이메일로 알림이 발송됩니다.
- CSO 매칭까지 완료되면 정산 조회 화면에 자사 데이터가 표시됩니다.
- 급한 경우 관리자(영업관리팀)에게 직접 문의하여 처리를 요청할 수 있습니다.

## 기타

### 화면이 깨져 보입니다

- 브라우저를 최신 버전으로 업데이트하세요.
- 권장 브라우저: Chrome, Edge (최신 버전)
- 브라우저 캐시를 삭제한 후 새로고침(`Ctrl + Shift + R`)하세요.

### 문의처

위 내용으로 해결되지 않는 문제는 관리자(영업관리팀)에게 문의하세요. 문의 시 아래 정보를 함께 전달하면 빠른 처리가 가능합니다.

- 사업자번호
- 발생한 문제의 화면 캡처
- 문제 발생 시점 (날짜, 시간)
- 사용 중인 브라우저 이름과 버전
