# 대시보드

CSO 정산 포털에 로그인하면 가장 먼저 표시되는 화면입니다. 홈 대시보드에서 최근 정산 현황을 한눈에 확인하고, 정산 조회 화면에서 상세 데이터를 검색할 수 있습니다.

## 홈 대시보드

로그인 직후 `/home` 경로에 표시되는 첫 화면입니다.

### 표시 정보

| 항목 | 설명 |
|------|------|
| 최근 정산월 | 가장 최근에 업로드된 정산 데이터의 처방월 |
| 총 정산 건수 | 해당 월의 전체 정산 건수 |
| 총 수수료 합계 | 해당 월의 수수료 총액 |
| 공지사항 | 관리자가 등록한 최신 공지 |

> **일반회원(CSO 업체)**은 자사와 매칭된 정산 데이터만 표시됩니다. 다른 업체의 데이터는 볼 수 없습니다.

## 정산 조회 화면

`/dashboard` 경로의 와이드 레이아웃 테이블 화면입니다. 정산서의 상세 데이터를 필터링, 검색, 정렬하여 조회합니다.

### 테이블 구성

정산 데이터는 최대 47개 컬럼으로 구성되며, 관리자가 `/admin/columns`에서 표시할 컬럼을 설정합니다. 일반회원에게 표시되는 주요 컬럼은 다음과 같습니다.

| 컬럼 | 설명 |
|------|------|
| 처방월 | 처방이 발생한 연월 (YYYY-MM) |
| 거래처명 | 의약품을 납품받은 거래처(병원, 약국 등) |
| 제품명 | 거래 대상 의약품 이름 |
| 수량 | 거래 수량 |
| 수수료 | 해당 건에 대한 CSO 수수료 금액 |

### 필터링

테이블 상단의 필터 영역에서 조건을 지정하여 데이터를 좁힐 수 있습니다.

- **처방월 선택**: 드롭다운에서 조회할 처방월을 선택합니다. 업로드된 처방월 목록이 자동으로 표시됩니다.
- **거래처별 필터**: 특정 거래처명을 선택하거나 입력하여 해당 거래처의 데이터만 표시합니다.
- **제품별 필터**: 특정 제품명을 선택하여 해당 제품의 정산 내역만 조회합니다.

### 검색

테이블 상단의 검색창에 키워드를 입력하면 거래처명, 제품명 등 텍스트 컬럼에서 일치하는 행만 표시됩니다.

### 정렬

각 컬럼 헤더를 클릭하면 오름차순/내림차순으로 정렬됩니다. 정렬 상태는 컬럼 헤더 옆의 화살표 아이콘으로 확인할 수 있습니다.

## 엑셀 다운로드

정산 조회 화면에서 현재 필터링된 데이터를 엑셀 파일로 내보낼 수 있습니다.

1. 원하는 필터 조건을 설정합니다.
2. 테이블 우측 상단의 **엑셀 다운로드** 버튼을 클릭합니다.
3. 현재 화면에 표시된 데이터가 `.xlsx` 파일로 다운로드됩니다.

> 다운로드되는 파일에는 현재 적용된 필터 조건에 맞는 데이터만 포함됩니다. 전체 데이터를 받으려면 필터를 해제한 후 다운로드하세요.

## 월별 합계

`/monthly-summary` 경로에서 월별 수수료 합산 데이터를 확인할 수 있습니다.

### 표시 항목

| 항목 | 설명 |
|------|------|
| 처방월 | 합산 기준이 되는 연월 |
| 총 건수 | 해당 월의 정산 건수 합계 |
| 총 수수료 | 해당 월의 수수료 금액 합계 |

### 정산 조회와의 차이

- **정산 조회** (`/dashboard`): 1행 = 1건의 개별 처방/거래 내역. 건별 상세 데이터를 확인할 때 사용합니다.
- **월별 합계** (`/monthly-summary`): 1행 = 1개월의 합산 데이터. 월별 추이를 한눈에 파악할 때 사용합니다.
